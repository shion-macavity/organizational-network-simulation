## ■ ターン 0
* **【初期思考】**
> さて、新しいゲームのアイデアが出た…これは結構ワクワクする！ 今回は本当に自由にやらせてもらえるみたいで、厳密なジャンルやゲームプレイのルールはなく、「自由なアイデア」を出すだけだっていう。それはすごく自由で、でもちょっと気が重いよね、わかる？ 私の部署は「SFゲーム企画」だから、それが自然な出発点になるけど、何でも取り入れてもいい、あるいは無視してもいいって言われた。つまり、宇宙船やエイリアンで縛られる必要はない、少なくとも新鮮なひねりを加えたいと思えばね。

私の主な目標は、しっかりとしたコンセプト、魅力的なストーリー、クールなコンテンツ、キャッチーなスローガンを作り出し、そして本当に重要なセールスポイントをしっかりと見極めること。 山本さんがまとめ出して提出してくれるから、私が思いつくものは彼女が扱えるものでなければならないし、それ以上に、本当に人々の心を掴むようなものでなければならない。

さて、どうアプローチするか…「固定観念にとらわれずに考えろ」って言われた。 私の仕事のスタイルはバランス型だから、創造性と、たとえ概念的にワイルドでも、少し地に足のついたものを探す傾向がある。 ただ突拍子もないものを思いつくのではなく、でも、あまりにも安全策に逃げるのも嫌だ。

最初のブレインストーミングでは、純粋な創造的なエネルギーに頼る必要がある。 たぶん、バラバラのアイデアを書き出し、普段は組み合わせないテーマを組み合わせることから始めるだろう。 例えば、ありふれた日常的な概念をスーパーSFの設定に入れる、あるいはその逆はどうだろう？ あるいは、よく知られたSFの定型を完全に逆転させるのはどうだろう？

最初に誰に相談するかというと、すぐに田中さんが思い浮かぶ。 彼女の仕事のスタイルは「直感的」で、この「固定観念にとらわれずに考える」という指示にはぴったりだと思う。 彼女なら、本当に斬新で型破りなアイデアを思いつくのが得意だろうし、私がバランスを取ろうとしすぎて見落とすかもしれないつながりを見抜けるだろう。 最初の、まだ未熟な考えを彼女にぶつけてみるだけでも、何か素晴らしいものが生まれるかもしれない。 実現可能性や構造をあまり気にせず、アイデアを出し合えばいい。 その生の、創造的なインプットが、この初期段階ではまさに必要としているものだ。

そして、いくつかのワイルドなコンセプト、大まかなストーリーのアイデアができたら、高橋さんを巻き込むかもしれない。 彼は経験豊富で、私の「バランス型」の仕事のスタイルと同様に、ワイルドなアイデアをより具体的なものに洗練させたり、潜在的な落とし穴や予期せぬ強みを示したりするのに役立つだろう。 どのようにして、本当に「型破りな」アイデアをゲームとして機能させられるかを見せてくれるかもしれない。

でも、まずは？ 田中さんとの、純粋で、何色もついていないアイデアセッションだ。 そう、それが計画だ。 まずはクレイジーなアイデアを流し出し、その後、それらをどのように形作るかを考える。
