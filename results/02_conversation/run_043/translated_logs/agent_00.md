私は、物理的なスポーツではないものを想像している。もしかしたら、「心のスポーツ」なのかもしれない。あるいは、創造性に基づいた競争なのかもしれない。例えば、「スポーツ」が、最も複雑なルーズベルト式機械を製作すること、あるいはライフシミュレーションの設定で最もユニークな魔法の庭を育成すること、あるいは、パズルを解くだけでなく、他人を挑戦させるためにパズルを*設計する*競争的なパズルゲームである場合を考えてみよう。

そこから始めてみよう。まずは、いくつかの初期段階の、突飛な考えを書き出す。自己検閲はしない。次に、高橋か田中と話してみたい。アイデアを出し合って、何がうまくいくか見てみよう。彼らは直感的だから、きっと理解してくれる。お互いのエネルギーに乗り、もし私のアイデアがファンタジー的なものに傾いている場合や、本当に新鮮な視点が必要な場合に、中村に連絡することもできる。それが良い計画だ。脳を活性化させて、創造的な人々と一緒にジャムしよう。
