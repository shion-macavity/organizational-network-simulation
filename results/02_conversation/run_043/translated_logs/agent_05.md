「このタスクにどうアプローチするか…」。最初のステップは、間違いなく自分自身のために、昔ながらのブレインストーミングをするだろう。頭の中を整理して、何が残るかを見極めたい。 「スポーツゲーム企画」の部署タグはあるが、彼らは明確に、その枠にとらわれなくても良いと言ってくれた。それは狭い檻ではなく、広い戦略的な方向性だ。解放される。

まず、思いつくキーワードをリストアップすることから始める。スポーツと、型破りな概念を混ぜ合わせるんだ。「スポーツ」は必ずしも肉体的なものでなくても良いのではないか？あるいは、精神や感情の競争なのか？あるいは、伝統的なスポーツを全く予想外の環境に置いたり、不可能なひねりを加えたりするのか？

物語については、ゲームで魅力的な語り口にするには何が必要かを考えている。勝ち負けだけでなく、旅、キャラクター、そしてリスクについてだ。スポーツに才能がないが、独自のやり方で優れているキャラクターはどうだろう？あるいは、スポーツが魔法や高度な技術と絡み合っている世界はどうだろう？

概念的には、新鮮なものを必要としている。単なる既存のものの繰り返しではない。アクション、RPG、FPS、レース、競技、パズル、ミステリー、協力、ライフシミュレーションなど、さまざまなジャンルから取り入れることができる…すごい数だ。おそらく、コンセプトに最も適したコアゲームプレイのスタイルを1つか2つ選び、すべてを詰め込もうとするのではなく、それに集中するだろう。たとえば、ライフシミュレーションで、「スポーツ」が予想外のこと、たとえば競技ガーデニングのようなもので、キャラクターの成長とスキルツリーのためのRPG要素がある。あるいは、犯罪を解決するために、奇妙な新しいスポーツの「物理法則」を理解する必要があるミステリーゲームだ。

いくつかの非常に大まかな概要ができたら——物語の種、コアゲームプレイのアイデア、そしておそらく緩い設定——誰に相談するかを考え始める。

「誰に最初に相談するか？」。私の内省的な性格を考えると、即座の直感的な反応を出すのではなく、深く掘り下げて物事を考えるのが好きな人を見つけたいだろう。直感的なアイデアは絶対に価値があるが、*私*の最初の相談相手としては、概念をさまざまな角度から検討し、建設的に欠点を指摘してくれる人が良い。

リストを見ると、渡辺さんも「内省的」なワークスタイルだ。私もそうであるように、部署の一般メンバーだ。私がいくつかの萌芽的なアイデアを持ったら、最初の詳細な議論をするのに最適な人だと思う。お互いに、より構造化された考えをぶつけ合い、生産的な方法で互いの仮定に挑戦できるかもしれない。チーム全体やリーダーに提示する前に、概念を洗練するための良い第一歩だと感じている。

まず一人でブレインストーミングを行い、その後渡辺さんに相談することで、コアアイデアをより強固に肉付けできると思う。その後、田中君や高橋さんなど、より直感的な思考の人たちを迎え入れ、概念をさらに押し上げるような、ワイルドで新鮮な視点を注入できるだろう。そして、リーダーである山本さんは、生の未形成な考えではなく、より洗練された一貫性のあるビジョンを提示したい人物だ。

計画はこうだ。まず一人で内省し、いくつかのコンセプトの種を開発する。次に、渡辺さんにアプローチして、より深く掘り下げて洗練させる。それが、堅実で方法論的なスタートだと感じている。
