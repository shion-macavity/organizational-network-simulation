## ■ ターン 0
* **【初期思考】**
> さて、新しいゲームのアイデアだけど…これ、まさに自分の得意分野だ！ 「自由なアイデア」「型破りな発想」が求められている、まさに理想的なブリーフ。最初から厳密な構造を設けず、創造性を自由に奔放にさせよう。我々は「現代設定ゲーム企画」を担当するけど、色々混ぜ合わせても良いと言われている、完璧だ。アクション、RPG、パズル、さらにはライフシム…可能性は無限大だ！

まず最初に、何か初期の火花を散らす必要がある。直感的思考の持ち主なので、普段はすぐ飛び込んで、何が出てくるか見てしまうタイプだ。最初から深く考え込まず、とにかくたくさんのアイデアを壁にぶつけて、何がくっつくか試すのが好きだ。頭の中にはすでにいくつか漠然としたアイデアが浮かんできている…例えば、現代社会の中に隠された世界のようなもの、あるいは古い神話を現代風にアレンジしたもの、あるいは日常的な出来事の中にファンタジー要素を盛り込んだもの、といった具合だ。例えば、バリスタが実はスパイネットワークを運営していたり、隣の猫が実は知性を持つAIだったりしたらどうだろう？ ハハ、たぶんありえないほど馬鹿げているけど、それがブレインストーミングの醍醐味だろう？

さて、これらの初期の生っぽい考えを誰にぶつけてみようか？ チームを見回すと…山本さんはリーダーで、「思慮深い」タイプだ。しっかりとした計画を望むだろう、それは後々には良いことだが、この最初の自由なアイデアの段階では、最高のスタート地点とは言えないかもしれない。田中さんと高橋さんは「バランス型」で、洗練と構造化が得意だが、まずは純粋でフィルターのかかっていないアイデアの創出をしたい。

そうだ！ 渡辺さんだ。彼もまた「直感的」なタイプだ。きっと私と同じように、頭の中にたくさんの奇抜なコンセプトが詰まっているだろう。二人で座って、ランダムな考えをたくさん出し合って、批判は一切せず、何かをクリックさせられるかどうか見てみればいい。きっとお互いを刺激し合い、未完成の考えを潜在的に素晴らしいものへと変えられるはずだ。それが、私の最高のアイデアが生まれる典型的な方法だ。静かな瞑想ではなく、弾丸のようなやり取りを通して生まれるのだ。

だから、まず最初に、渡辺さんに話を聞きに行こうと思う。ざっくばらんなブレインストーミングセッションだ。何も形式化する必要はなく、ただ歯車を回して、どんなクレイジーなコンセプトを一緒に作り出せるか見てみたい。もしかしたら、本当に斬新なものに偶然出会えるかもしれない。

アプローチとしては、渡辺さんに「最近、何か面白いゲームのコンセプトを考えていることはあるか？ 現代設定で、何かひっかかっていること、ぜひ探求してみたいと思っていること、でもちょっとひねりを加えたようなものは？」と、簡単な質問から始めるだろう。それから、自分の未完成の考えを「例えば、郊外の自宅で秘密裏に地下抵抗組織を建設する、居心地の良いライフシムはどうだろう？」「あるいは、手がかりがすべて難解なインターネットミームに隠されている、刑事ゲームはどうだろう？」といった、突拍子もないことをぶつけて、話のきっかけを作ろう。そこから、コアコンセプトを考え、次にストーリーを考え、それに合ったゲームプレイを検討していく。それが私のやり方だ。クールな初期のフックから構築していくのだ。
