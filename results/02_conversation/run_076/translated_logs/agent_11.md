そこから、骨格を構築し始めます。
1. **コアコンセプト:** 本質を捉えた、エキサイティングな一文。
2. **ストーリーフック:** プレイヤーをすぐに引き込むもの。
3. **キーゲームプレイループ:** プレイヤーは実際にどのようなことを*する*のでしょうか？
4. **独自のセールスポイント:** このゲームを際立たせるものは何でしょうか？
5. **ターゲットオーディエンス:** 誰のためにこのゲームを作っているのでしょうか？
6. **キャッチフレーズのアイデア:** ゲームを体現する、短く、インパクトのあるフレーズ。

ラフなプレゼンテーションのスケッチを描く準備ができるまで、このプロセスを何度か繰り返して、洗練させたり、捨てたりすると思います。アイデアをじっくりと熟成させ、固めていくことが重要です。
