# エージェント個人史(和訳): Tanaka (ID: 3)

## ■ ターン 0
* **【初期思考】**
> ふむ、もう新しいプロジェクトですか？しかも、本当にただ… *考える* だけ、と。制約もなく、何でもあり。実年齢を考えると、これは結構気分転換になりますよ、ね。時々、いつものルーティンに飽き飽きするんですけど、白紙の状態というのは、わくわくしますね。

私の部署は「終末世界ゲーム企画」ですが、何でも取り入れられる、あるいは完全に逸脱してもいいと言われました。「固定観念にとらわれず考えろ」と。それは耳にタコができますね。私はいつも、最高のアイデアは、心を自由にさまようままに、最初から厳格に決めつけないことから生まれると思っています。

それで、「まず誰に相談するか」「どのようにこのタスクに取り組むか」…

まず、私がどのように取り組むか。通常、私は頭の中にイメージや感情が浮かんでくるのをただ待つだけです。それは火花のようなもので、それに従って進んでいきます。スプレッドシートから始めるわけではありませんよ、ね？「もし～だったら？」とか「～だったら面白くないか？」といった感じです。私の直感が、アイデアに将来性があるかどうか教えてくれます。

だから、まずは一人で少しブレインストーミングをします。心を「終末世界」に働かせつつ、それをひねることも考えます。もし、暗いのではなく、希望に満ちた世界だったら？もし、終末が実は…奇妙なことに良いことだったら？あるいは、予想外の方法で再建する世界だったら？アイデアをスケッチしたり、ある種の雰囲気に合うメロディーを口ずさんだりするかもしれません。まず、ゲームの核となる*感情*を掴む必要があります。メカニクスだけでなく、感情的な魅力、雰囲気です。

いくつかの強い火花、物語の粗いアイデア、一般的なコンセプト、あるいはいくつかキャッチーなフレーズが浮かんできたら、*その時*になって誰かに相談することを考えます。

そして「誰に」？そうですね、私たちの部署の誰もがかなり…内省的です。それは私の直感的なスタイルとは良い対照になります。彼らは、穴を見つけたり、論理を見つけたり、詳細がきちんと成立するようにするのが得意です。私はワイルドでエキサイティングなコンセプトを思いつくかもしれませんが、彼らが「でも、それはどうやって*実現する*の？」とか「プレイヤー体験はどうなる？」と尋ねてくるでしょう。

だから、私はいくつかの*異なる*初期コンセプトを彼らに提示すると思います。ただ一つではなく。例えば、「この新しいプロジェクトのために、いくつか考えていることがあるんです。皆が植物になる終末世界、あるいは『終末』が単に皆がテクノロジーの使い方を忘れて昔ながらの方法に戻る世界、というアイデアが浮かんだんです。どう思いますか？何か興味を引くものはありますか、それとも全く的外れですか？」

私は完全に練り上げられた計画を持って行くわけではありません。生の、直感的な閃きを持って行き、彼らの内省的な性質がそれを地に足の着いたものにするのを手伝ってくれるか、あるいはさらに予期せぬひねりを加えてくれることを期待します。彼らは、私の初期の、おそらく未熟なインスピレーションを、強力なストーリー、魅力的なコンテンツ、そして明確なセールスポイントを備えたフルゲームに発展させる可能性を判断するのに役立つかもしれません。彼らの視点は、私の初期のインスピレーションをより具体的なものに変えるために不可欠です。
