## ■ ターン 0
* **【初期思考】**
> さて、新しいゲームプロジェクト、ですか。かなり自由度が高いですね、それはワクワクする一方で少し気が重いところもあります。私の部署は「現代設定ゲーム企画」ですが、思いのままに進めても構わないと言われました。他の要素を取り入れたり、完全にジャンルを無視したりしても良いそうです。それは良いですね、クリエイティブな制約はほとんどありません。アクション、RPG、FPS、レース、対戦、パズル、ミステリー、協力プレイ、ライフシミュレーション…基本的に何でもありです。ここが、私の内省的な側面を活かせる場所です。

まず最初に、私が本当にクールで新鮮だと思うゲームの種類を理解する必要があります。23歳なので、たくさんのゲームをプレイしてきました。だから、古いアイデアを単に焼き直すだけにしたくありません。しっかりとした時間をかけてブレインストーミングし、考えを自由にさまよう必要があります。コーヒーを飲みながら、紙やドキュメントにアイデアを書き出すような、そんな時間を過ごすんです。判断もフィルタリングもせず、とにかくすべてを書き出します。どんなストーリーに引き込まれるだろうか？ 現代設定でうまくできていないユニークなゲームコンセプトは何か？ あるいは、さらに良いのは、「現代」の概念を捻ってみることです。

私は内省的なタイプなので、会議で未熟なアイデアを一方的にぶつけることは避けたいと思っています。ストーリー、コアコンセプト、実際のゲームプレイの内容、キャッチーなスローガンなど、アイデアのしっかりとした基盤を構築し、誰かに話す前に、主要なセールスポイントをしっかりと固めたいと思っています。そうすることで、議論する際には、一貫性のあるビジョン、共有し、構築できる具体的なものを持てるようになり、漠然とした考えだけではなくなります。

ある程度自信を持てる良い内部ドラフトができたら、それを誰に相談するかを考え始めます。私の部署内には、さまざまな人がいます。経験豊富な佐藤さんと渡辺さんがいて、そして私と同じように内省的な作業スタイルを持つ伊藤さんがいます。それから、佐藤さんと渡辺さんよりも若く、私よりもほんの少し年上のリーダーの山本さんがいます。

最初のアイデアとしては、まず伊藤さんと簡単なカジュアルな会話をしてみることにしています。私たちはどちらも内省的ですから、彼もよく考え抜かれたアイデアを、混沌としたブレインストーミングよりも高く評価してくれるかもしれません。彼は私と同じように、一般的なメンバーという立場にもあります。私たちは一緒にアイデアを洗練させたり、他の人に相談する前に、彼から正直なフィードバックを得たりできるかもしれません。それは、気軽に試す良い方法です。

その後、佐藤さんや渡辺さんに話すことを検討するかもしれません。彼らの経験は、業界全体やオーディエンスの視点など、私が考慮していない貴重な視点を提供してくれるかもしれません。彼らは多くのことを見てきましたから、私のアイデアの欠点を指摘したり、さらに良いことに、アイデアを強化する方法を提案してくれるかもしれません。

そして、もちろん、リーダーの山本さんがいます。最終的には、彼が顧客のためにすべてをまとめるので、私は必ず彼に自分のアイデアを提示する必要があります。しかし、彼にアイデアを提示する前に、できるだけ磨き上げたいと思っています。

他の部署については、興味深い考えです。山本さんは彼らのリーダーとつながっているので、その道筋はあります。しかし、この最初の段階では、自分のコアアイデアを固めるために、まずは自分の部署内で集中することにします。強力なコンセプトができたら、*その時*、他の部署からの外部の視点を得ることのメリットがあるかどうかを検討します。特に、私のアイデアがクロスジャンルの可能性を秘めていて、彼らが取り組んでいることと一致する場合や、本当に新鮮で偏りのない視点が必要な場合に検討します。しかし、今のところは、頭を下げて、深く考え、記憶に残る何かを作り出すことに集中します。
