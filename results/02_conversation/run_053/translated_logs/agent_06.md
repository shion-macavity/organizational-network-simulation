初期コンセプトの良いプールができたら、残りのチームメンバーを迎え入れます。佐藤さん、田中さん、伊藤さん、高橋さんは、皆、熟考型です。彼らは、生のアイデアを分析し、可能性を見出し、課題を特定し、より具体的な形に発展させるのが得意です。佐藤さんの経験は、突飛なアイデアを現実的に落とし込んだり、独自の視点からさらに発展させたりする上で、非常に貴重になるでしょう。田中さんと伊藤さんは、物事を構造化し、論理的な流れを考えるのが得意です。高橋さんは、まだ若いですが、熟考型で、新しい世代のプレイヤーにとって何が機能するかについて、新鮮で分析的な視点を提供してくれるかもしれません。

ですから、私のアプローチは二段階プロセスになります。

第一段階：まず、鈴木さんと渡辺さんとカジュアルな会話をしたり、小規模なミーティングをしたりします。私たちが持っている完全な自由度を説明し、「間違いのないアイデアはない」というルールを強調します。彼らに、いつもの軍事/ウォーゲームの快適ゾーンから完全に抜け出すように考えてもらいます。ストーリー、コンセプト、独自のメカニズム、あるいは印象的なイメージや感情など、思いつくものをスケッチしてアイデアを出し合ってもらいます。まだ洗練されたアイデアは必要ない、生の素材だけが欲しいということを理解してもらいます。彼らに数日かけて、自由に考え、思いつくことを書き留めてもらうでしょう。

直感的な人たちに幅広くアイデアを出し、熟考型の人がそれを洗練させて構築することで、古いパターンにとらわれず、本当に革新的でよく考えられたものを生み出すことができる最高のチャンスになると思います。これはエキサイティングな挑戦であり、皆が何を作り出すのか楽しみにしています。
