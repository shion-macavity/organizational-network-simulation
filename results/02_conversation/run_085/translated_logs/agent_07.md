## ■ ターン 0
* **【初期思考】**
> さて、新しいゲームのアイデア、完全にゼロからだ。ストーリー、コンセプト、コンテンツ、キャッチコピー、セールスポイント…そして、彼らは「自由なアイデア」と「型破りな発想」を特に強調している。これは実にワクワクするな、実際。この年齢になると、最高のアイデアは、時々、自分の心を少しさまようことから生まれるということを学ぶ。

まず最初に、自分の考えを流し始める必要がある。私の仕事のスタイルはバランスが取れているので、すぐに飛び込むこともないし、麻痺するほど考えすぎもしない。ここでは、ジャンルやメカニクスを気にせず、ゲームに呼び起こしたいテーマや感情をざっくりと書き出すことから始めるのが良いアプローチだと思う。ただ、一般的なムードだ。新鮮な感じ、わかるだろう？

我々の部署が「スーパーヒーローゲーム企画」であることを考えると、それに沿うか、ひねるか、完全に無視するかを検討すべきだろう。今回の指示には「他の要素を取り入れたり、そうしないことも歓迎する」とあるので、かなり自由度が高い。派手なスーパーヒーローではなく、彼らの*人間性*を探求するのはどうだろうか、あるいは「超能力」がはるかに些細で日常的なものであればどうだろうか？あるいは、完全に逆転させて、静かな観察や美しい非暴力的な世界での複雑なパズル解決など、まったく異なるものにするのはどうだろうか。可能性は本当に無限だ。

さて、誰に最初に相談しようか？特に「型破りな発想」のためのブレインストーミングには、 ‘直感的’な仕事スタイルの人々と話すのが本当に有益だと思う。彼らは、他の人に何か新しいものを呼び起こすことができる、予期せぬアイデアを思いつく才能を持っていることが多い。

リストを見ると…小林さんと加藤さんが ‘直感的’な仕事スタイルだ。加藤さんは若いので、私の視点とはまったく異なる視点を持っているかもしれない。これは多様なアイデアを生み出すのに最適だ。小林さんも経験豊富だが、彼の直感的なスタイルは、初期の考えをぶつけ合うのに適している。

まずは、自分自身で少しだけ独りブレインストーミングをして、いくつかの基礎となるアイデアを書き出すことから始めようと思う。それから、おそらく加藤さんに声をかけるだろう。彼女は若く、業界に長くいないため、固定観念にとらわれにくい、新鮮なアイデアが生まれることが多い。彼女がコーヒーを飲んでいる時など、気軽に声をかけて、「加藤さん、この新しいゲームのアイデアのために、本当に型破りなことを考えようとしているんです。何か初期の突飛な考えはありますか？たとえ馬鹿げているように見えても。」と言ってみるだろう。私は彼女に完全に形になったコンセプトを話すのではなく、彼女の想像力を私と一緒に自由に羽ばたかせるように誘いたい。お互いの自発的な考えをぶつけ合えるだろう。

その初期の直感的なアイデアの爆発の後、おそらく小林さんに連絡を取るだろう。彼も直感的だが、経験年数が長いため、いくつかの突飛なアイデアを地に足の着いたものにするか、別の方向に発展させるのに役立つかもしれない。もちろん、その後、いくつかの粗いコンセプトができたら、分析と洗練を手伝ってもらうために、吉田さんに意見を求めることになる。そして最終的には、私たちのリーダーである山口さんに話を聞いてもらい、最終的に彼女にすべてをまとめる手伝いをしてもらう必要がある。

しかし、今のところ、最初のステップは自分の脳を温め、次に加藤さんから新鮮な直感的な火花を得ることだ。この初期段階では、自分自身や他の人を検閲してはいけない。面白いものを何を見つけられるか見てみよう！
