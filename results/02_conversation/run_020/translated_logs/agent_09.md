# エージェント個人史(和訳): Kato (ID: 9)

## ■ ターン 0
* **【初期思考】**
> 新しいゲームコンセプトの企画が出た！これは楽しみだ。「自由なアイデア」「固定観念にとらわれない」「他の要素を取り入れる」…そういうブリーフ、大好きだ。僕の部署は「ミリタリー/ウォーゲーム企画」だけど、全くそれにこだわらなくてもいいって言われた。素晴らしい。またいつもの戦争ゲームなんて作りたくないんだ。もちろん、作れるんだけど、面白いに決まらないだろう？もうすでに、色々な方向性が頭の中を駆け巡っている。

まず最初に思い浮かんだのは、とにかくフィルターなしで、頭の中を自由にさまようこと。他にない、本当にユニークなものを生み出したい。軍隊をテーマにすることはできなくもないけど、どうにか捻りを加えられないか？例えば、武器で戦うのではなく、外交や、あるいはガーデニングで戦うゲームはどうだろう？あるいは、軍隊を舞台に、戦後の復興に焦点を当て、生活シミュレーションとコミュニティ作りを重視するゲーム。あるいは、宇宙戦闘のために設計された宇宙船を舞台にしたパズルゲームで、パズルは宇宙船の生命維持装置に必要な電力を再配線すること。乗組員は全員、愛らしいエイリアンだ。わかるだろう？これが、オープンエンドなタスクが好きな理由だ。

実際に始めるにあたって、まずはこれらの突飛なアイデアをたくさん吐き出す必要があると思う。自己検閲はしない。どんなに突飛に聞こえても、とにかく書き出す。そして、いくつかの明確なコンセプトができたら、少しずつ肉付けを始める：どんなストーリーが合うか、コアとなるゲームループは何か、フックは何か。

誰に相談しようか…リストを見ている。山口さんが部署のリーダーだから、最終的には彼女にプレゼンするだろうけど、未加工のアイデアをいきなり持って行くのは避けたい。少し洗練させてからだ。中村さんと佐々木さんは、どちらも思慮深いので、批判的なフィードバックやコンセプトの洗練には最適だけど、最初の、本当に自由なブレインストーミングの段階では、僕と同じように直感的な人を探した方がいいかもしれない。

それで、小林さんが思いついた。彼女も直感的で、経験も豊富だから、創造的に考えるための経験がたくさんあるはずだ。彼女こそ、本当に突飛なアイデアを気軽にぶつけられる、最高の相手だと思う。お互いの突飛な考えをぶつけ合い、すぐに却下することなく、発展させていける。彼女に「小林さん、新しいコンセプトの企画で、思いっきり自由にやるように言われたんだけど、頭がいっぱいなんだ。少しブレインストーミングに付き合ってくれない？悪いアイデアはないよ、とにかく壁にぶつけて、何がくっつくか見てみよう」って言うつもりだ。

まずは、軍隊のテーマを完全に無視して、プレイヤーにどんな感情を呼び起こしたいかを考えてみることから始めよう。そして、その感情の中に、型破りな方法で軍隊の要素を注入してみる。例えば、深く感情的なストーリーを、静かで忘れられた戦争の背景に置き、戦うのではなく、真実を発見することに重点を置く。あるいは、協力型のゲームで、基地を管理するが、戦うのではなく、広がる生態学的災害から絶滅危惧種を守るために資源を調整する。資源配備に「戦術」的な要素を取り入れる。考えただけでワクワクする。

やっぱり、まずは小林さんに相談しよう。お互いに刺激し合えるはずだ。そして、本当に独創的なコンセプトがいくつかできたら、中村さんや佐々木さんのような、より思慮深く批判的な視点から、それが実際に実現可能で楽しめるかどうかを判断してもらい、コアコンセプトが明確になったら、キャッチコピーやセールスポイントについても考え始める。でも、まずは純粋で、何もないところから生まれる創造性だ！
