# エージェント個人史(和訳): Suzuki (ID: 1)

## ■ ターン 0
* **【初期思考】**
> 新しいプロジェクトのブリーフが届いた。新しいゲームのデザインを、ストーリー、コンセプト、コンテンツ、キャッチコピー、主要なセールスポイントを含めて考える。ジャンルやゲームプレイのスタイルは自由で、私たちの部署は「スポーツゲーム企画」とはいえ、完全に自由なアイデアを求めている。これはかなり面白い。制限がない、ってことだね？ 普段は「サッカーゲームを、でももっと良くする」とか、「バスケットボールシムをどうやって革新するか？」みたいな話ばっかりだけど、今回は完全に白紙の状態だ。

僕のワークスタイルは「バランス型」だから、頭に浮かんだ最初の突飛なアイデアをすぐに提案するのではなく、でも、考えすぎて身動きが取れなくなるのも避けたい。創造的なブレインストーミングと、それを形にするための構造的な思考のバランスが重要だ。

まず最初に、少し時間をかけて、自分の頭の中を自由にさまよう必要があるだろう。どんなゲームを*作りたい*んだろう？ 何が新鮮だろう？ スポーツゲーム企画の部署だから、思いもよらないひねりを加えたスポーツゲーム、あるいはスポーツゲームとは全く関係ないけど、スポーツゲームにあるような競争やスキル要素を取り入れたゲーム、というのはどうだろう？

考えてみよう… スポーツではないスポーツゲーム、みたいな。例えば、ファンタジー世界を舞台にした「スポーツ」はどうだろう？ あるいは、夢を叶えるために架空のスポーツで秀でなければならないライフシムはどうだろう？ もっと突飛なことを言うなら、アスリートが駒であるパズルゲームで、彼らの動きを戦略的に配置してポイントを獲得する、みたいなのはどうだろう。いや、それは少し抽象的すぎるな。

よし、リセット。 「固定観念にとらわれない」。スポーツの*感覚*はどうだろう？ チームワーク、ライバル意識、個人の成長、ドラマ、スペクタクル。それらの感覚を、伝統的なスポーツに縛られずに活かすことができる。

だから、最初の段階として、今日は数時間、ブルー・スカイ・シンキングに集中する時間を取りたい。判断はせず、思いつく限りの突飛なアイデアをメモ帳に書き出す。ストーリーのフック、奇妙なコンセプト、うまくいかないかもしれないけど、面白そうなゲームメカニクス。

いくつかの粗いコンセプトができたら、誰に話を聞くかを考え始めることができる。

同僚を見ていると…

リーダーの山本さん。31歳で「直感的」タイプ。まとめと提出は彼女が行うので、最終的には最高のアイデアを彼女に相談する必要がある。直感的だから、型破りなアイデアを高く評価してくれるかもしれないが、彼女がすぐに理解できるように、明確に説明する必要がある。漠然としたアイデアをそのまま持ち込むのではなく、ストーリー、コンセプト、コアゲームプレイについて、少なくともいくつかの基礎的な考えは持っておくべきだ。

次に、佐藤さんと高橋さん。どちらも「直感的」タイプだ。佐藤さんは53歳、高橋さんは40歳。初期の生っぽいアイデアを気軽に相談するのに最適だ。直感的な人は、コンセプトをすぐに理解し、それを発展させたり、可能性を見抜いたりする才能を持っていることが多い。どちらか、あるいは両方に簡単な相談をすれば、混沌とした初期の考えを洗練させたり、全く新しい方向性を見つけたりするのに役立つかもしれない。また、何が*魅力的*に感じるかを判断するのにも役立つだろう。

渡辺さんは43歳で「熟考型」。より具体的なコンセプトがあるものの、穴を見つけたり、実現可能性（技術的なことではなく、コンセプトの面で）、コンテンツをより論理的に構造化したりする必要がある場合に最適だ。熟考型の人は、ストーリーが意味をなしているか、コンセプトが首尾一貫しているか、セールスポイントが強くて明確に表現されているかを確かめるのに役立つ。ワイルドなアイデアを、本当に実現可能で魅力的に聞こえるものに変えてくれるだろう。

だから、ゲームプランはこうだ。
1. ソロでのブレインストーミングセッション。多様で制約のないアイデアをいくつか書き出す。
2. いくつかの生きた素材ができたら、佐藤さんか高橋さんに気軽に相談する。いくつかの最も突飛なアイデアを提示して、彼らの意見を聞き、感情レベルで何が機能し、何が機能しないかを明確にする。リスクが低く、何がうまくいくかを明確にするのに役立つだろう。
3. その後、少し洗練されたコンセプトがいくつかできたら、渡辺さんに相談する。彼女の「熟考型」のワークスタイルは、すべての角度、潜在的な課題、価値をどのように表現するかを考慮することで、良いアイデアを素晴らしいアイデアに変えるのに最適だ。
4. 最後に、十分に練り上げられたアイデアがあれば、 Yamamotoさんに簡単な打ち合わせをスケジュールして、詳細なデザインに乗り出す前に、彼女の賛同とフィードバックを得る。彼女が提示するのだから、彼女に響く必要がある。

これはバランスの取れたアプローチだ。自分の創造性を活かし、素早い直感的なフィードバックを得て、より分析的な視点からそれを洗練させ、最後にリーダーの承認を得る。そして、自由なことにワクワクしている！ どんな狂ったアイデアが思いつくか見てみよう。宇宙を舞台にした競争的な料理ゲーム、あるいは、ハイステークスの競争的なガーデニングシム、というのはどうだろう？ 可能性は無限だ！
