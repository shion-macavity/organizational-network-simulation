## ■ ターン 0
* **【初期思考】**
> さて、どうしようか。新しいゲームプロジェクトで、無料のアイデアから、物語、コンセプト、内容、スローガン、セールスポイント…と、とにかく「型破りな発想」を強く求めているようだ。それは楽しみだ！ 私のような歳になると、色々なトレンドが行ったり来たりするのを見てきたから、たまに「型破り」なアイデアが、ただ新しいペンキを塗った古いアイデアのリサイクルにしか感じないこともある。でも、本当に新鮮なものが現れることにはいつも期待している。

私の部署は「ファンタジーゲーム企画」だが、それはあくまで広範な戦略的な方向性だと彼らは言っている。無視してもいいし、他の要素を取り入れても構わない。それは良いことだ。ファンタジーはそもそも大きな傘だから。ハイファンタジー、アーバンファンタジー、ダークファンタジー、歴史ファンタジー、あるいは完全に抽象的なものでもできる。そしてゲームプレイのスタイルは完全に自由だ。アクション、RPG、パズル、ライフシム…すごい、自由度が高いな。

さて、まずは最初に、自分の考えを整理する必要がある。誰かと話す前に、いくつかの初期コンセプトをじっくりと考えてみたい。どんな「ファンタジー」が本当に私を惹きつけるのだろうか？ あるいは、どんな「非ファンタジー」がファンタジーの要素と面白く融合できるだろうか？ ゲームを本当に魅力的にするものは何か、数時間プレイして忘れ去るのではなく、ずっと付き合いたいと思わせるものは何か、それを考えている。良いストーリーは重要だが、ゲームプレイにもフックが必要だ。

このアプローチについてだが…私は一般メンバーで、部署リーダーの山本君がすべてをまとめ上げる。だから、私の役割は、しっかりとした、よく考えられたアイデアを貢献することだが、求められた場合は自由にブレインストーミングすることもだ。私の「バランスの取れた」仕事スタイルは、最初のアイデアに飛びつくのではなく、あらゆる角度を考慮するのが普通だ。

さて、誰に相談すべきだろうか？

まず最初の一歩は、自分の考えを自由にさまよう静かな時間を持つと思う。頭に浮かんだどんな突飛なアイデアでも書き留める。まだ何も検閲はしない。ただ、最初の創造的なエネルギーを湧き上がらせるだけだ。ユニークなメカニクス、魅力的な物語、そしてゲームがプレイヤーにどのような感情的な影響を与えたいのかを考える。

その最初のブレインストーミングの後、自分の部署の誰かと連絡を取りたいと思う。田中さん、あるいは渡辺さん、かな。田中さんは50歳で、私の世代に近いし、彼の「直感的」な仕事スタイルは、私のより構造化された考えと面白い予期せぬつながりをもたらすかもしれない。あるいは渡辺さん、私たちはお互いと同じ年齢で、同じく「バランスの取れた」仕事スタイルを持っているから、共通点を見つけやすく、実用的でありながら創造的な方法で互いのアイデアを築き上げることができるかもしれない。高橋さんは思慮深いので、生の断片ではなく、より発展したアイデアを熟考してくれるだろう。

最終的には、私たちのリーダーである山本君とも話したいが、最初からすぐにはおそらくない。彼に提示する、たとえ粗削りでも具体的なアイデアをいくつか持っておきたい。ただ「何をすべきか？」と聞いているだけではないことを、彼に積極的に貢献していることを示したい。彼は若くて、私は彼に指示を待っているだけではないことを示したい。

もう一つの部署は興味深いポイントだ。「自部署では入手できない情報を獲得するか、提供することで、社内では話しにくいことを表現したり、聞いたりできるかもしれない」。それは強いヒントだ。もし私のアイデアが停滞し始めたら、あるいは社内チームの議論の流れを乱すことなく、本当に突飛なコンセプトを試したい場合、そこにいる誰かに連絡を取ることは非常に役立つだろう。任意だが、念頭に置いておくべき良いカードだ。ただし、まずは自分の部署に集中する。

さて、フェーズ1：個人的なブレインストーミング。フェーズ2：自分の部署の同僚との非公式な議論で、洗練し、拡大する。そして、よりまとまりのあるものになったら、山本君と話し合う準備ができる。これは、私の仕事スタイルと同じように、バランスの取れたアプローチだと感じられる。
