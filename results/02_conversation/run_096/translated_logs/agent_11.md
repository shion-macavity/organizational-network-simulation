## ■ ターン 0
* **【初期思考】**
> よし、頭をフル回転させよう。新しいゲームのアイデアか。しかも「自由なアイデア」ベースとなると、それはもう耳にタコができるほど嬉しい。直感的なスタイルには特に相性が良いからね。最初から厳密なルールに縛られるわけではないのも素晴らしい。

まず最初に、誰に相談しようか？　思いついたことをそのまま書き出すのは、いつもそうしているやり方だけど、今回はゲームのコンセプト全体を練り上げるプロジェクトだから、早い段階で別の視点を得たり、大まかな構想ができたらアイデアをぶつけられる相手がいればいいなと思っている。

チームメンバーを見ると… 山口さんはリーダーだから、最終的には必ず共有する必要がある。でも、生で、フィルターのかかっていないアイデアを最初に話す相手としては、そうではないかもしれない。彼女は熟考型で、もう少し構造化されたものを好むだろう。それは後から構築していくことにする。小林さんと吉田さんも熟考型だから、後でアイデアを洗練させたり、見落としを指摘してもらうのに適任かもしれない。佐々木さんは「バランス型」だ。これは面白いかもしれない。私の突飛なアイデアについていけるし、同時に現実的な視点も提供してくれるかもしれない。初期のアイデアがまとまったら、後でコーヒーでも飲みながら話してみよう。

でも、最初の相談相手は？　正直、まっさらな状態だと、最初は自分自身と少し向き合いたい。どんな物語が生まれてくるかを感じ取りたいんだ。私の頭は、他の声を取り入れる前に、自由にさまようことを許した方がうまく働くんだ。だから、まずは自分自身に相談して、直感的な火花を散らすと思う。

このタスクにどう取り組むか、というところだけど… 「自由なアイデア」、「SFゲーム企画」、「そして、その先へ」。よし、「SF」の部分は良い出発点だ。でも、それに縛られる必要はない。自由に組み合わせればいい。頭の中ではすでに可能性がざわめいている。

まずはキーワードやコンセプトをブレインストーミングするだけだ。大きなホワイトボードに書き出して、批判は一切しない。SFの面白い要素ってなんだろう？　宇宙旅行、AI、異星人との遭遇、ディストピア、ユートピア、タイムトラベル、遺伝子工学、サイバネティクス… それから、それらをひねってみる。もし古代の異星文明が技術ではなく、感情を残していったとしたら？　あるいは、未来社会が完全に仮想現実の中で生きているが、その外の世界は、忘れ去られた幻想的な魔法の領域だったら？

ゲームのスタイルについても考える。SFなら、必ずシューターでなければならないか？　そんなことはない。宇宙コロニーを管理するライフシムはどうだろう？　でも、それは社会的なダイナミズムや、隔離された生活の心理的な影響に焦点を当てたものだ。あるいは、異星の言語を解読するパズルゲームで、パズル自体が物語や世界観に組み込まれているもの。

まずは、新鮮でエキサイティングな、核となるコンセプトを見つける。それから、その周りに物語を構築していく。フックは？　登場人物は？　何が危険にさらされているのか？　そこから、環境、メカニズム、ユニークな機能などの具体的な内容は自然に生まれてくる。しっかりとしたコンセプトと物語の構想ができたら、キャッチーなスローガンやセールスポイントを作り出すのはずっと簡単になる。人々の心を掴む、記憶に残るものにしたい。

だから、私のやり方はこうだ：最初の段階では一人でブレインストーミングをして、アイデアを自由に流れ込ませる。そして、それなりのコンセプトが集まったら、それを洗練させて、本当に残るものは何かを見極める。その時、佐々木さんに声をかけて、ちょっとしたおしゃべりで初期のフィードバックをもらおうかな。でも、今は自分自身、白いキャンバス、そして無限の可能性の世界だけだ。これが楽しい部分なんだ！
