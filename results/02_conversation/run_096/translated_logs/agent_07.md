# エージェント個人史(和訳): Nakamura (ID: 7)

## ■ ターン 0
* **【初期思考】**
> よし、頭をフル回転させよう。新しいゲーム、テーマは「自由な発想」「型破りなアイデア」…これはまさに私の得意分野だ。55歳という年齢で、色々なアイデアを見てきたが、良いゲームの核となるものは常に新鮮で、人を引き込む何かだ。そして、直感的な仕事のスタイルなので、細かく計画を立てるよりも、まず飛び込んでみて、何がうまくいくか試したい。

課題は、ストーリー、コンセプト、コンテンツ、キャッチコピー、セールスポイントだ。SF部門向けだが、ジャンルを自由に組み合わせられるのは素晴らしい。宇宙船やレーザーにばかり囚われたくない。

まず最初に考えたのは、誰かにアイデアをぶつけてみることだ。ブレインストーミング、つまり、生で、フィルターなしの思考を、判断せずに。誰が良いだろうか？

同僚を見ていると… 山口はリーダーで、思慮深い。物事を洗練させるのは得意だが、最初の、混沌とした創造性の爆発には向かないかもしれない。小林と吉田も同じように思慮深く、もう少し構造化されたアプローチを好むだろう。今の私にはそぐわない。

しかし、加藤は25歳で、同じく直感的だ。若い直感的な頭脳こそが必要なのだ。彼はきっと私とは全く違う視点を持っているだろう。それは「型破りな発想」に最適だ。お互いに刺激し合えるはずだ。奇抜なアイデアを否定するのではなく、「もっと伸ばしてみたら？」と、さらに突き放してくれるだろう。そういうエネルギーでこのプロジェクトをスタートさせたい。

どうアプローチすべきか？ まず、一人で座って詳細な提案書を書こうとするのは絶対に避ける。今、創造的なプロセスを阻害するだけだ。まずは、生のエッセンスを抽出する必要がある。SFでありながら、単なる宇宙シューティングゲームではない、本当にクレイジーなアイデアをいくつか書き出してみよう。例えば、宇宙規模のガーデニングをテーマにしたゲームはどうだろう？あるいは、意思を持つ惑星を舞台にした探偵物語？本当に予想外のものを。

いくつかの荒唐無稽なアイデアができたらいよいよ加藤に話を聞く。こう言うだろう。「加藤、新しいゲームのコンセプトで、思いっきり暴走するよう言われているんだ。制限はない。SF部門だが、自由に解釈できる。いくつかアイデアが浮かんできたけど、君がこっそり作りたかった、とんでもなくクレイジーなアイデアを聞かせてくれ。少しアイデアを出し合って、何かが火花を散らすか見てみよう。」

直感的でいる私にとって、これが一番良いスタート方法だと思う。生のエッセンスを手に入れてから、それをどう形作るかを見ればいい。そして、特に若い直感的な仲間と一緒にそれをやる？完璧だ。
