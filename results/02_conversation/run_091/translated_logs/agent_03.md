## ■ ターン 0
* **【初期思考】**
> さて、完全にゼロからゲームのコンセプトを考え出す必要があるんだ。ストーリー、コンセプト、内容、スローガン、そしてセールスポイント。しかも「現代設定ゲーム企画部」向けの仕事だけど、正直、思いっきり自由に、あらゆるジャンルを組み合わせても構うよって言われている。それは結構ワクワクするな、制限も特にない。僕の役割はアイデアを出すだけで、確定させるんじゃないから、それはありがたい。リーダーの山本さんが、顧客へのまとめをするんだ。

まず最初に、自分だけで時間をかけてブレインストーミングするのがいいだろう。ちょっと頭を空っぽにして、自由に連想させてみる。すぐに完璧なものを生み出す必要はない。そもそも*僕*がどんなゲームが好きだろう？どんな物語に心を動かされるんだろう？今の市場に何が足りないんだろう？「現代設定」って、色々な意味がある。現代の日常、ハードな都市ドラマ、未来的なテクノロジー・スリラー、あるいは現代社会へのコメントを別のジャンルで表現したような、もっとさりげないものまで。

最初はキーワードやテーマを書き出すことから始めようか。現代社会における繋がりや断絶について、とか？あるいは、日常に隠された魔法やミステリーについて？それがクールな切り口になりそうだ。ひねりのあるライフシミュレーションゲーム、とか？現代の課題を乗り越えていく、という強い物語性を持ったパズルゲーム、とか？

いくつか大まかなアイデアが浮かんできたら、誰に話を聞くかを考えないといけない。この初期の創造的な爆発には、新しい、あるいは型破りな考え方にもオープンな人に相談したい。突飛なアイデアをすぐに却下するのではなく、むしろそれを発展させたり、新しい視点を提供してくれるような人だ。

同僚を見ていると、伊藤さんは僕と同い年で、「直感的」と記載されている。ブレインストーミングには良さそうだ。今のトレンドや、新鮮で面白いことについて、同じような波長で感じているかもしれない。気軽にアイデアを出し合って、批判される心配もないだろう。何についてでも話せるはずだ。週末に何をプレイしたか、たまたま見た映画、といったことからゲームのアイデアに突入してもいい。この初期の創造段階には、あまり硬直的でない人がいるといい。まだ未熟なコンセプトをすぐに山本さんに持って行くのは避けたい。彼はリーダーだし、もっと構造化されたアイデアを必要としているだろう。もう少し土台ができたら、佐藤さんや渡辺さんに経験と広い視野から意見を聞くのもいい。でも、創造的なエネルギーを湧き上がらせるには、伊藤さんが最初の目的地にぴったりだ。

だから、僕のアプローチはこうだ。
1.  **個人でのブレインストーミング:** 僕、ペン、ノートだけ。自己検閲はしない。何でも自由に書き出す。テーマ、ユニークなゲームメカニクス、潜在的な物語、そしてそれらが「現代」の文脈にどのように適合するかを考える。プレイヤーにどんな感情を体験してもらいたいかについても考える。
2.  **伊藤さんとのカジュアルな会話:** いくつかのかなり大まかなコンセプトができたら、伊藤さんに声をかける。たぶん、「この『自由にアイデアを出す』プロジェクトに取り組んでいるんだけど、頭の中が色々な方向に飛んでいて。ちょっとだけ、僕のランダムな考えを聞いてもらえないかな？君の考えも聞けたら嬉しいな。特に素晴らしい必要はないし！」みたいなことを言うだろう。そうすれば、お互いに思いつくことを自然に共有し、初期のコンセプトを洗練させ、発展させることができる。とにかく、協力的なエネルギーを湧き上がらせることが重要だ。
