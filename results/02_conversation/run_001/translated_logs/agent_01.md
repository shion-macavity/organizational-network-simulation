## ■ ターン 0
* **【初期思考】**
> さて、部門のために全く新しいゲームのアイデアを考え出す必要があります。ストーリー、コンセプト、コンテンツ、キャッチコピー、セールスポイント…すべてです。そして、これは「Superhero Game Planning」を名乗ってはいますが、「型にはまらない」ものでなければなりません。正直、それは素晴らしい挑戦です。これほど自由を与えてくれるのは嬉しいです。

私の仕事のスタイルは「バランス型」で、創造的で自発的なアイデアと、それらを開発するためのより構造化された論理的なアプローチの両方を重視します。ですから、このプロジェクトでは両方の良いミックスが必要だと思います。

まず、個人的なブレインストーミングセッションから始めると思います。思いつく限りのどんな突飛なアイデアでも、どれほどばかげているように見えても、書き留めるだけです。「スーパーヒーロー」は広いテーマですが、それを本当にひねることができます。もし、戦いがないとしたらどうでしょうか？もし、「超能力」が普通の仕事で役に立つような、ありふれたものだったらどうでしょうか？あるいは、スーパーヒーローであることの*社会的*側面、アクションではなく、それについてだったら？あるいは、自分を明かさずに力を使い犯罪を解決するミステリーゲームだったら？典型的な「悪者を殴り倒す」という物語を超えていきたいのです。

ある程度のラフなコンセプトができた後、それらを洗練する必要があります。ここが私の「バランス型」が活きてくる部分です。魅力的なゲーム体験の可能性を秘めたアイデアはどれで、フルコンセプトに肉付けできるかを判断する必要があります。

さて、*誰に最初に相談するか*ですが…最初の、本当に自由なブレインストーミングでは、誰かにアイデアをぶつけられると素晴らしいと思います。それは、即座の実用性にあまりこだわらない、自然な直感力のある人です。佐藤と高橋は、まさに「直感的」な仕事のスタイルなので、それに当てはまります。佐藤は少し若いため、私の中に何かを呼び起こすような、本当に斬新で型破りな視点を持っているかもしれません。高橋も直感的で、私の年齢に近いので、創造的なエネルギーを補完してくれるでしょう。

おそらく、まず佐藤にアプローチします。彼は普段、かなりオープンで熱心です。コーヒーでも飲みながら、この新しいプロジェクトに対する最初の「クレイジーなアイデア」がないか、さりげなく聞いてみたいと思います。最終的な決定をするのではなく、創造的な思考を刺激し、私たちの直感がどのように一致したり、興味深い方向に分岐したりするかを見るのが目的です。「もし、スーパーヒーローゲームを、家賃を払って身分を隠しながら、時々超スピードを使って通勤時間を短縮するだけのライフシムにしたらどうでしょうか？」というような、初期の未熟な考えをいくつか彼と共有して、彼の反応を見て、彼の頭の中がどこへ向かうか見てみたいのです。

そして、いくつかのより強力なコンセプトが開発されたら、おそらく、渡辺や山本（私たちのリーダー）のような「内省的」な人に相談したいと思います。渡辺は、私が始めたコンセプトをより詳細に批判的に見て、山本に提示する前に潜在的な弱点や改善の余地を特定するのに役立つかもしれません。彼女は、論理的な流れや、さまざまな要素が実際にプレイ可能なゲームにどのように組み合わさるかを考え出すのに役立つでしょう。

しかし、まずは、それらの型破りな、フィルターのかかっていないアイデアを外に出しましょう。ただ考え、お絵かきをするための静かな時間を作らなければなりません。これは楽しみになるでしょう。
