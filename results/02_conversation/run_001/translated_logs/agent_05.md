## ■ ターン 0
* **【初期思考】**
> ふむ、なるほど、新しいゲームプロジェクトか。面白そうじゃないか、アイデアを自由に広げられる良い機会だ！ 「自由なアイデア」「ジャンル自由」、本当にそうなんだろうか？ そんなブリーフは大好きだ、チェックすべき項目も、守るべき古い型もない。

私の部署は「スーパーヒーローゲーム企画」で、他の要素を取り入れたり、取り入れなかったりと言われていますが、スーパーヒーローというジャンル自体に、まだまだ開拓されていない可能性が眠っていると思います。ただ悪役を殴るだけじゃない、ってこと？ スーパーヒーローの*生活*はどうだろう？ あるいは*結果*？ それとも、メインのヒーローではなく、その周囲にいる人物を演じるのはどうだろう？ ああ、もう頭の中がアイデアでいっぱいだ！

考えているのは… 最も強く、最も速いことが重要なのではなく、もっと… 人間的な何かはどうだろう？ 若くて派手なヒーローではなく、すべてを見てきた年配のヒーローで、次の世代を指導しようとしながら、自分の過去と闘っている、そんなヒーローはどうだろう？ あるいは、ヒーローではなく、彼らのイメージを管理したり、秘密基地を運営したり、あるいは足を洗おうとする悪役を演じるゲームはどうだろう？ それは本当に面白そうだ。

最初のステップとして、何かを具体化しようとする前に、思いつくどんな突飛な考えでも書き留めておこうと思う。 ただの頭の中の放出だ。 フィルターはかけない。 そういう風に私の最高のアイデアが始まることが多いんだ、ただ思いつくままにそれを吐き出すんだ。 まずは責任、犠牲、あるいは力の重荷といった、核となる感情的なテーマについて考えるだろう。 そして、メカニズムやストーリーの展開を重ねていく。

本当に生で、わくわくするようなコンセプトがいくつかできたら、まず最初に話したいのは高橋さんだ。 彼女は私と同じように直感的で、アイデアの野生的な可能性を見抜いたり、私が思いつかなかった別の創造的な層を付け加えたりする才能があることが多い。 ただアイデアを出し合って、プレッシャーもなく、何がうまくいくかを見るだけだ。 時には、自分のアイデアを誰かに声に出して聞くことで、詳細にこだわらずに洗練させることができるんだ。 お互いにぶつけ合い、本当にユニークなものを生み出すことができるだろう。

その後の高橋さんとの初期の創造的な爆発の後、田中さんや渡辺さんなど、より思慮深い目で持ちこたえるかどうかを確認したり、山本さんには全体的な戦略的な視点から意見を聞いたりするかもしれないが、まずはそうはしない。 私の直感は、とにかく創造し、創造し、創造し、後で磨くことを心配するように言っている。

アプローチ1：独りでのブレインストーミング、自由連想、制限なし。
アプローチ2：高橋さんと創造的な打ち合いで、初期の直感的なアイデアを拡大・洗練させる。

さて、ヒーローが*選ばれる*のではなく、たまたま力を手に入れ、それからどうすればいいのかを考えながら、完全にありふれた仕事を持ち続ける、そんな物語はどうだろう？ 例えば、中年のおじさんが突然スーパーパワーを発揮するが、オフィスでの政治や締め切りでいつもストレスを感じているときにだけ、というような。 そしてゲームは、秘密のヒーローとして街を救いながら、解雇されないようにバランスを取るという内容だ。「The Accidental Hero's Double Life（誤ってのヒーローの二重生活）」というコンセプトだ。 内容は、スーパーヒーロー的な行為のための「エネルギー」バーと、プロのパフォーマンスのための「エネルギー」バーという、2つの別の「エネルギー」バーを管理することになる。 ミッションは、昼休みの間に軽犯罪を阻止したり、誰も疑わないように、しつこい上司を「スーパー叫び」で黙らせたりすることを含む。 主要なセールスポイントは、英雄主義に対する関連性が高く、ユーモラスな視点、人生シミュレーションとスーパーヒーローアクションのユニークなブレンド、そして最もありそうもない状況でアンダードッグが成功する喜びだ。 スローガンは？ 「世界を救うのは大変。 仕事を救うのはもっと大変。」 ああ、気持ちいい！ すでにアートスタイルが見えてくる… 90年代のコミックストリップのような、レトロな雰囲気だが、現代的なひねりが加えられている。 ええ、これはぜひ探求したいと思っている。
