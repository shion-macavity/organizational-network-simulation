「エポック・エコー」や「クロノス・ネクサス」といった名前はどうだろう？
**物語**は、主要な歴史上の人物をあらゆる時代から引き裂き、まったく未知の世界に送り込む「時間異常」と呼ばれる謎めいた宇宙現象を中心に展開する。これは終末後の地球ではなく、奇妙な動植物や、おそらく古代の非人間的な遺跡がある、まったく新しい惑星だ。
クレオパトラ、レオナルド・ダ・ヴィンチ、織田信長、マリー・キュリー、ユリウス・カエサル、さらにはアラン・チューリングや忘れ去られた有名な作家など、指導者、発明家、哲学者、芸術家の組み合わせが、当初は混乱し、自分の時代に戻ることができない。彼らは、この新しい世界で共存し、理解し、最終的には時間異常の真実と、その中で彼らの目的を明らかにしなければならない。それは試練か、避難場所か、それとも監獄か？

**コンセプト**は、協力型サバイバル、社会シミュレーション、そして壮大なミステリーのブレンドとなる。プレイヤーは単一のキャラクターを操作するのではなく、これらの歴史的アイコンが創り出す新たな社会に影響を与える。それは、多様な精神が、非常に異なる社会構造や技術に慣れ親しんでいるにもかかわらず、異質な環境でゼロから新たな文明を築き、それぞれのユニークなスキルと知識を活用してどのように協力（または対立）するかというものだ。ダ・ヴィンチが原始的な資源を使って発電源を設計しようとし、マリー・キュリーが惑星の奇妙な元素について理論を立て、織田信長が地元の野生動物に対する防御を組織化する様子を想像してほしい。

**内容**としては、ゲームプレイには以下が含まれる可能性がある。
1.  **探索:** 新しい世界のバイオーム、資源、遺跡を発見する。
2.  **基地建設/都市計画:** 居住地を設計・管理する。歴史上の人物によって、それぞれの既知の特性に基づいたユニークなボーナスや課題が発生する（例：カエサルは組織化に優れているが、より多くの権力を求めるかもしれない）。
3.  **クラフト＆研究:** 異なる時代の知識を組み合わせて、新しい技術を開発する。火薬の大砲と蒸気機関を組み合わせたり、異質な結晶から初期のコンピュータ回路を設計したりすることを想像してほしい。
4.  **社会ダイナミクス:** 歴史上の人物間の関係を管理する。ライバル関係、同盟、ロマンス、さらには民主的な選挙や専制的なクーデターが発生する可能性がある。対話の選択肢と道徳的ジレンマが重要になる。
5.  **ミステリーの進行:** 古代の遺跡、異質な言語の解読、さらには失われた記憶の断片を組み合わせることで、時間異常に関する手がかりを明らかにする。
6.  **「エコー」能力:** 各歴史上の人物は、元の力や影響のユニークな「エコー」を持ち、それがこの新しい世界で特別な能力として現れる。例えば、将軍に対する戦術的ブーストや、科学者に対する研究速度の加速など。

**キャッチコピー**は、「歴史を書き換え、運命はまだ書かれていない」や、「時を超え、新たな世界が待っている」といったものになるだろう。後者の方が、より神秘的で魅力的だと思っている。

そして、**セールスポイント**は？
1.  **前例のない歴史の融合:** あらゆる時代から象徴的な人物と、まったく新しい舞台でプレイし、対戦する。
2.  **深く、戦略的な社会シミュレーション:** 多様な歴史的思考の衝突と協力を体験する。

4.  **無限のプレイアビリティ:** プレイごとに異なる同盟が結成され、技術が優先され、異なる道が選ばれる可能性がある。
