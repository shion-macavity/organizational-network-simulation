それから、Post-Apocalyptic 部署の Sato がいる。興味深い。インタラクションは必須ではないが、全く異なるジャンルの視点を取り入れることで、「固定観念にとらわれず、発想しなさい」という要件を本当に満たせるかもしれない。もし、スポーツと終末世界の設定を何とか組み合わせたらどうだろう？サバイバルスポーツ？あるいは、文明の廃墟から生まれるスポーツ？それは本当にユニークなアングルになるかもしれない。ただし、彼女は思索的なタイプなので、生半可なブレインストーミングには向かないだろう。自分の部署のテーマに触れる萌芽的なアイデアができあがってから、彼女に相談する方が良いかもしれない。

私の直近のアプローチは、直感的な仕事スタイルを考えると、とにかくひたすらアイデアを出し始めることだ。すぐに何かを構造化しようとするつもりはない。ただ、たくさんのランダムな考え、キーワード、シナリオ、さらにはくだらないコンセプトを書き留めるだけだ。何も検閲しない。とにかく、たくさんのアイデアをプールする。そして、それなりに散らかってから、つながりやパターン、あるいは本当に際立っているものを見つけ始める。

その後の Kobayashi との初期ブレインストーミングの後、アイデアの中に終末世界の要素があると感じたら、Sato と話すことを検討するかもしれない。あるいは、単に「スポーツ」の考え方に染まっていない、全く新しい視点が必要な場合に、彼女が良いかもしれない。しかし、今は純粋で、未加工のアイデア生成に集中することだ。本当に新鮮なものを創り出そう。これは楽しみだ！
