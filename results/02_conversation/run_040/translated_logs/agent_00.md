## ■ ターン 0
* **【初期思考】**
> よし、頭を整理しよう。さて、私たちの部署「終末世界ゲーム企画部」のために、全く新しいゲームのアイデアを出すのが目標だ。ストーリー、コンセプト、コンテンツ、キャッチコピー、セールスポイント、全てだ。そして、重要なのは、典型的な終末世界ものにとらわれず、箱の外から本当に考え、どんなゲームプレイ形式でも構わないということだ。それは実にワクワクする。

佐藤として、内省的な性格なので、まず皆でブレインストーミングセッションに飛び込むのではなく、少しの間、自分の頭の中でアイデアをじっくりと煮詰めることを好む。じっくりと発酵させるんだ。 「終末世界」と「型破りな発想」を組み合わせたときに、私にとって「終末世界」とはどういう意味なのかを考える必要がある。必ずしも暗くて陰鬱なものなのか？もしそうでないとしたら？再生について、荒廃の中に美を見出すことについて、あるいは、非日常的な状況下での意外な日常性について、どうだろうか？

したがって、私の最初のステップは、今日のうちに、あるいは明日早朝に、静かな時間を作り、ノートを使って漠然としたコンセプトを深く掘り下げることだ。まず、ジャンルの典型的な要素を書き出し、次に、それぞれをねじ曲げたり、裏切ったりするよう試みるだろう。例えば、資源の枯渇ではなく、*予想外の豊かさ*について、突然変異体ではなく、*退屈*が最大の脅威であるという設定はどうだろうか？あるいは、終末は壊滅的な出来事ではなく、ゆっくりと忍び寄るものであり、今、私たちはその「その後の世界」に生きているだけ、という可能性もある。

いくつかの非常に粗く、高レベルなコンセプト——詳細な計画ではなく、単なる種——が、面白く、真にユニークであると感じたら、その時、誰に相談すべきかを考え始める。

誰に最初に相談すべきか？私の仕事のスタイルを考えると、私と同じように思慮深いアプローチを取る人に惹かれるだろう。すぐに高橋さんが思い浮かぶ。彼女も内省的で、61歳という年齢から、私の萌芽的なアイデアを地に足の着いたものにするか、あるいは拡張するのに役立つ、豊富な経験と異なる視点を持っている。彼女は多くのゲームコンセプトを見てきたはずだから、潜在的な落とし穴を見抜いたり、コンセプトをさらに発展させたりするための洞察は非常に貴重だろう。私は完成したアイデアを持って彼女のところに行くのではなく、いくつか検討している方向性を示し、単に「終末世界ゲームに、このような雰囲気は合うと思いますか？」とか、「このアイデアを検討していますが、率直な感想を聞かせてください」と尋ねるだろう。

山本さん、私たちのリーダーには、アイデアを少し洗練させてから相談するつもりだ。彼は要約をする人だから、初期段階でも、よく検討されたものを提供したい。鈴木君は素晴らしいが、コンセプトに少し形がついてから、若い視点を得るために彼に相談するかもしれない。田中さんは直感的だが、抽象的なものを投げつけて、明確な「直感」反応を引き出せるように、自分自身で少し土台を築く必要があると感じている。

そして、別の部署の中村さん、スポーツゲーム企画部の。それは面白い考えだ。おそらく、しっかりとした、ほぼ完成したコンセプトができ上がってから、もっと後になるだろう。外部の視点、特に全く異なるジャンルの人から得られる視点は、アイデアが本当にユニークで独自のものなのか、それともニッチすぎるのかを本当に際立たせてくれるかもしれない。しかし、この初期の、内省的な段階では、そうではない。まず自分の考えを整理する必要がある。
