まず、プレイヤーにどのような*感情*や*体験*をしてほしいのかを考えます。英雄的な気分にさせたいのか、思慮深い気分にさせたいのか、挑戦させたいのか、リラックスさせたいのか？この感情的な核は、ジャンルを最初に選ぶよりも、ゲームプレイのスタイルや物語をより効果的に決定することが多いです。

次に、当部門の「歴史ゲーム企画」のバックグラウンドが、アイデアにどのように影響を与えたり、*ひねりを加えたり*するのかを検討します。私たちは歴史的な要素が得意なので、歴史的な設定やテーマを取り上げ、完全に予想外で非歴史的なひねりを加えることができるかもしれません。たとえば、歴史は私たちが思っているようなものではないとしたら？あるいは、普通の歴史上の人物がひそかに並外れた能力を持っていたとしたら？あるいは、有名な出来事に焦点を当てるのではなく、語られなかった物語、日常生活、あるいは歴史が隠してきたかもしれない*魔法*のような潜在的な力に踏み込むとしたら。それは本当に斬新なアプローチになるでしょう。

RPGやミステリーゲームがよく得意とする、深い物語を可能にするようなものを考えています。しかし、メカニクス面でも魅力的なものも必要です。おそらく、歴史的な時代を舞台にしたミステリーゲームに、選択と結果の強い要素を取り入れ、プレイヤーの行動が文字通り隠された歴史を書き換えたり、明らかにしたりするようなもの。

いくつかの大まかなコンセプトができたら、基本的なストーリーの前提、コアとなるゲームループのアイデアを練り上げ、それが何がユニークなのか、つまり主要なセールスポイントを考えます。スローガンは、本質を本当に掴んだ後で、おそらく最後に決めるでしょう。

誰に最初に相談するか？という点ですが、内省的な性格なので、誰かに急いで話すことはしません。少なくとも2つか3つの、ある程度発展したアイデアを頭の中に持っておきたいと思っています。アイデアを共有する準備ができたとき、おそらく高橋さんに相談します。彼も内省的なので、よく考えられたコンセプトを高く評価し、より深い洞察を提供したり、論理的な欠陥を特定するのに役立ってくれると思います。私たちは、急ぐことなく、創造的な可能性を深く掘り下げることができます。山本さん、リーダーに話すのは、より洗練され、提示できる状態になった後で、とっておきます。そして、高橋さんと話し合った後、特定のアイデアが外部の視点から恩恵を受ける可能性があると感じた場合、よりアクション指向のゲームに焦点を当てている他の部門の誰かに、歴史的なミステリーを単なる物語を超えて、どのように*アクティブ*にしたり、*魅力的に*したりするかについてフィードバックをもらうことができるかもしれません。しかし、それはさらに下流の話です。まずは、私とノートだけです。
