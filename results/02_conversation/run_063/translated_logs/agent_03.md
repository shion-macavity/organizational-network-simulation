## ■ ターン 0
* **【初期思考】**
> さて、新しいプロジェクトの概要が出た。「物語、コンセプト、内容、スローガン、主要なセールスポイント」を設計するんだ。そして、我々が「歴史ゲーム企画部」に所属しているにも関わらず、完全に自由に、型破りな発想をすることを求められている。面白い。いつもと違うことを奨励してくれているのは良いことだ。

よし、まずは「誰に相談すべきか」「どのようにこのタスクに取り組むべきか」だ。

正直なところ、まず自分自身でアイデアを書き出すのが最初の考えだ。この業界に長くいると、他の人と早合点してしまうと、まだコアコンセプトを練り上げる前に水が濁ってしまうことがある。アイデアを自分の頭の中で少し煮詰めて、どんな奇妙な方向に進もうとしているかを見るのが、私のいつものやり方だ。この「自由なアイデア」という方針は、まさにそれを後押ししている。

どうやって取り組むか？まずは、頭に浮かんだランダムなキーワードやテーマを書き出すところから始めるだろう。必ずしも歴史関連とは限らず、ただゲームとして面白そうに聞こえるものなら何でもいい。例えば、夢やパラレルワールド、あるいは本当にユニークなコアメカニズムについて。それから、その周りに物語を紡ぎ、コンセプトを開発し、自然にそこから生まれてくるコンテンツを見ていく。スローガンとセールスポイントは、コアアイデアが固まってから考える。

何かが形になり、実際に*提示*できるような、具体的なものができたら、誰かに相談することを考える。部署内なら、ワタナベあたりだろうか。彼女もバランスの取れた働き方をしていて、40歳という年齢で良い経験も積んでいる。創造的なアイデアを受け入れつつ、より現実的で実践的な視点を得るための良い相談相手になるかもしれない。サトウやイトウも、思慮深く洞察力に富んだ意見を持っているかもしれないが、深入りする前に、初期の迅速なフィードバックが必要な場合もある。

いきなり、真っ白な状態のままヤマモト部長に持って行くことはしない。彼は全てをまとめ上げる人だから、しっかり練り上げられたコンセプトを持っていく必要がある。

他の部署との連携…それは面白い選択肢だ。緊急用にと、とりあえずポケットに入れておこう。壁にぶつかったり、努力にも関わらずアイデアがいつもの部署のやり方と*あまりにも*似ていたりする場合、歴史ゲーム以外の視点を得ることが、状況を変えるかもしれない。しかし、今は自分の頭の中で、最初の創造的なアイデアを自由に広げることに集中しよう。外部からの制約がないままに、本当に「型破りな発想」をするための最良の方法だと感じている。

よし、ステップ１：一人でブレインストーミングをする。コーヒーを一杯飲んで、静かな場所を見つけて、ただ自分の心を自由にさまよう。どんな*楽しい*ゲームをプレイしたいか？どんな物語を体験したいか？何が浮かんでくるか、見てみよう。
