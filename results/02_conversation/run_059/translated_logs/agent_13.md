## ■ ターン 0
* **【初期思考】**
> さて、頭を動かしてみよう。新しいプロジェクト、新しいアイデア、制限はない…まさに私が好む挑戦だ。スポーツゲームの企画が私たちの部署の役割だが、今回の指示では、何でも取り入れ、あるいは「スポーツ」の部分を無視することも許されている。素晴らしい。考えるべき枠組みはなく、ただ純粋な創造性だけだ。

まず、「誰に最初に相談するか」「どのようにこのタスクに取り組むか」だ。部署リーダーとして、そして私自身も直感的思考の持ち主として、私の直感は、最初から状況を一新する必要があると言う。形式的な会議でじっと座っていても、火花は飛び散らないだろう。

「どのようにこのタスクに取り組むか」という初期の考えは、真にオープンで自由な発想環境を育むことだ。「スポーツゲームの企画」部署に期待されることを忘れなければならない。重要なのは「自由なアイデア」と「固定観念にとらわれない思考」だ。既存のスポーツゲームの漸進的な改善は求めていない。人々が「へえ、そう思ったことがない！」と叫ぶような、全く新しいものを求めている。

そのため、まず最初に、特定の誰かだけに形式的に相談するのではなく、少人数の多様なグループを集めて、本当に多様な思考を促したい。型破りなアイデアを臆することなく投げ出せる人だけでなく、多角的に物事を考えられる人も必要だ。

「誰に最初に相談するか？」。もし一人だけ、気軽に話を聞いて、初期の生きたアイデアを試すために選ぶとしたら、おそらく中村だろう。彼は私と同じように直感的で、概念を素早く理解し、それを発展させたり、新しいアイデアを点火させたり、既存のアイデアを洗練させたりするのに役立つ、即座でフィルターのかかっていない反応を提供する才能がある。彼は比較的若いため、トレンドや若いプレイヤーが何を求めているかについて、新鮮な視点を持っているかもしれない。例えば、「ねえ中村、少し時間ある？新しいプロジェクトについて考えているんだけど、指示は『固定観念にとらわれない』んだ。今まで夢見た中で、最もワイルドなゲームコンセプトは何だ？ジャンルは完全に無視して？」といった軽い会話だ。これは、自分自身のために、アイデアを出す準備をするためのウォーミングアップに過ぎない。

しかし、実際のデザインプロセスの*開始*にあたっては、「どのようにこのタスクに取り組むか」は、小規模でダイナミックなチームで進めることになる。具体的には、中村、加藤、そして山田を招集したい。中村と加藤は私と同じように直感的で、早期に足踏みを踏むことなく、素早くアイデアを出し合える。山田はバランスが取れているため、思考の流れを妨げることなく、より深く考えるための適切な質問をし、抽象的な概念に完全に迷子になるのを防ぐことができるだろう。

彼らにこう伝えるだろう。「皆さん、新しいプロジェクトだ。スポーツゲームについて知っていることはすべて忘れてしまえ。ジャンルも忘れてしまえ。私たちの使命は、ゲームの定義を書き換えるほどユニークで、魅力的で、キャプティベーティングなゲームコンセプトを思いつくことだ。ストーリー、コンセプト、コンテンツ、キャッチフレーズ、セールスポイント…すべてだ。私は皆さんに、家に帰って、白昼夢を見たり、ブレインストーミングをしたり、何が必要でもして、想像できる最もクレイジーで、最も説得力のあるアイデアを持って戻ってきてほしい。この初期段階では、どんなアイデアも、あまりにばかばかしい、ありえない、あるいは「スポーツではない」ものであっても構わない。自分たち自身や他の人を驚かせよう。」

おそらく、会議室のような堅苦しい場所ではなく、コーヒーやカジュアルなランチをしながら、非常に非公式な「アイデア出し」セッションをスケジュールするだろう。目的は何も最終決定することではなく、どんなに突飛なものであっても、すべての考えをテーブルに出すことだ。後で洗練させて形作っていくのは問題ではない。今は、できるだけ広く網を張ることが重要だ。このアイデアの要約は*私の*責任であることを強調し、プレゼンテーションではなく、純粋なアイデア出しに集中してもらうようにする。そうすることで、彼らの個々の強みと視点をすぐに引き出すことができる。
