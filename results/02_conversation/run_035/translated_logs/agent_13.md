部署リーダーとして、そして思索的な人間として、私の最初の直感は、すぐに大規模なブレインストーミングセッションに飛び込むことではない。まず、自分自身でこの概要をしっかりと消化し、本当に理解する必要がある。「自由な発想」は両刃の剣だ。創造性にとっては素晴らしいが、初期の構造や、その自由の*意図*に対する理解がないと、ただ混沌とした混乱に終わってしまう可能性がある。

では、「誰に最初に相談すべきか？」「どのようにこのタスクに取り組むべきか？」

このような場合に私の社内プロセスとしては、チームを招く前に、少し静かに熟考することが多い。まずは、ただ*考える*ことから始めるつもりだ。私にとって「犯罪と捜査」とは、広義の意味でどういう意味だろうか？探偵や殺人事件だけではないはずだ。あらゆる種類の真実を明らかにする、謎を解き明かす、人間の行動を理解する、あるいは犯罪組織を構築したり、それを阻止したりすることかもしれない。「捜査」の部分は、科学的、歴史的、あるいは人間関係的なものであっても良い。そして、「固定観念にとらわれず考えろ」という言葉を信じれば、伝統的な犯罪に限定する必要はないかもしれない。

概要の*精神*を一人でじっくりと考えてから、次に論理的なステップとして、チームを巻き込むと思うが、必ずしも正式な会議ですべてを一度に集める必要はない。我々が持つ多様なワークスタイルや経験を考えると、それらを効果的に活用したい。

おそらく、まずチーム全体に簡単な社内メッセージを送るだろう。プロジェクト概要を説明し、「自由な発想」を強調し、ジャンルやゲームプレイの柔軟性、そして「固定観念にとらわれず考えろ」という励ましを伝える。すぐに完成したコンセプトを求めるわけではない。代わりに、それぞれが少しの時間、せいぜい一、二日、思いつく限りの初期の考えやキーワードを書き出すように頼む。プレッシャーはかけない、悪いアイデアはない、ただ生のインプットを求める。彼らにとって、「犯罪と捜査」とは、いつもの制約を無視するように言われたとき、どのような*感覚*だろうか？どのようなストーリー、キャラクター、またはメカニズムが、この極度の自由の中で彼らの興味を引くのだろうか？

こうすることで、誰のワークスタイルに関わらず、他の人に影響されずに、あるいは即座のグループディスカッションのプレッシャーを感じることなく、それぞれの初期の考えを練り上げることができる。思索的な私と同じように、中村は時間をありがたく思うだろう。直感的なスタイルを持つ Kato と Sasaki は、本当に奇妙で素晴らしい初期の火花をもたらすかもしれない。バランスの取れた Kobayashi と Yoshida は、この初期段階でも、いくつかの基礎となるアイデアや考慮すべき実用的な側面を提供するかもしれない。そして、我々の一番若い Yamada は、トレンドや革新的なものに対する新鮮な視点をもたらすかもしれない。

それらの初期の個人の考えを収集した後、誰とより焦点を絞った会話を始めるべきかを考えるだろう。生のアイデアをレビューし、パターンが現れたり、本当にユニークな火花があるかどうかを確認する。その結果、 Kato や Sasaki のような直感的な人を、本当に抽象的なアイデアを拡張するのを手伝ってもらったり、Kobayashi や Yoshida のようなバランスの取れた人を、よりワイルドなコンセプトを地に足の着いたものにするのを手伝ってもらったり、あるいは Nakamura を、より深く構造化された議論のために取り上げたりするかもしれない。

他の部署にすぐに相談することはない。概要にはオプションとして記載されているが、まず*我々の*部署が独自の初期の方向性と明確なアイデアを確立することが重要だと感じる。我々にとって「自由な発想」とは何を意味するのかを定義する必要がある。より具体的な内部コンセプトが得られたら、新鮮な視点を得るため、あるいはある程度発展したアイデアを試すために、他の部署とのやり取りが価値があるかもしれない。しかし、今は、内部の歯車を回し、我々のチーム内で幅広いオプションを生成することだ。

要するに：
1. 私自身の概要に対する熟考。
2. チームに初期の、生の、プレッシャーのない考えやキーワードを求める一般的な依頼を送る。
3. チームの初期のインプットをレビューする。
4. その後、有望なアイデアを具体化するために、特定のチームメンバーとのより焦点を絞った議論を決定する。
