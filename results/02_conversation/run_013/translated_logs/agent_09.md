さて、誰に相談するのが最初でしょうか？ 実は、最初に思い浮かぶのは、ペンと紙を片手に一人で時間を過ごすことです。ただブレインストーミングをして、どんなにばかげた考えでも、頭に浮かんだら書き留めるだけです。そうすることで、頭の中がクリアになり、時には意外なつながりが明らかになることがあります。個人的に魅力的に感じる体験、ゲームを本当にユニークにするもの、そして流行を追いかけるのではなく、新鮮なものをどう提供できるかについて考えるでしょう。スポーツゲームの経験から、自然と競争、チームワーク、成長、スキルについて考えることになるでしょうが、これもまた、その枠を超えたチャンスであることを知っています。

しかし、もし私が自分の考えを完全に整理する前に相談する人物を選ばなければならないとしたら、何か意外なものを引き出してくれるかもしれない人物となると、中村を頼ると思います。彼は直感的な仕事ぶりで、私よりもずっと若いため、彼の視点は私とは大きく異なる可能性が高いです。彼が、私が考えもしなかった全く突飛なことを言い出すかもしれませんし、時にはそれが、いつもの思考パターンを打ち破るために必要なことなのです。構造化された議論を期待するのではなく、どんなに荒唐無稽な初期の種を蒔けるかを見るために、ただ弾丸トークのようなブレインストーミングセッションになるでしょう。おそらく、「中村、部長から新しいゲームの完全に自由なアイデアを出す許可が出たんだ。最近、プレイヤーが求めているような、スポーツ以外のコアな体験について、どんな突飛な考えがある？」と切り出すと思います。

タスクそのものへのアプローチ方法としては、中村との初期のアイデア発想以外には、間違いなく自分のデスクに戻ります。まず、ジャンルに関係なく、長年楽しんできたさまざまなゲームメカニズムをリストアップします。次に、それらを基本的な物語や概念的な枠組みに織り交ぜてみます。強力なコンセプトが鍵だと信じています。すべてを結びつけるものが必要です。自分自身に問いかけます。私たちが販売するコアなファンタジーとは何か？ どんな感情を呼び起こしたいのか？ 挑戦、驚き、連帯感、リラックス？

スポーツの経験と全く異なるものを組み合わせようと思います。たとえば、競争的なスポーツ要素をミニゲームとしてではなく、キャラクターの成長やコミュニティの構築の中核部分として織り交ぜたライフシミュレーションゲーム。あるいは、「能力」が異なる競技の分野に基づいたアクションRPGで、単なる直接的な戦闘ではなく、独自のやり方で使われるものです。プレイヤーが、戦士や冒険者ではなく、独特の才能を持つ個性的な個人を集めてチームを作り、それらの才能を予期せぬ協調的な方法で目標を達成するゲームを想像しています。単にレースで勝つだけでなく、弓使いの正確さ、重量挙げの力、体操選手の敏捷性など、全員が協力して解決しなければならない複雑な環境パズルを解くのです。

いくつかの大まかなコンセプトができたら、具体的なコンテンツについて考え始めます。どんな世界、キャラクター、挑戦が合うでしょうか？ 何がリプレイ可能にするでしょうか？ 次に、それをキャッチーなスローガンといくつかの主要なセールスポイントに絞り込みます。スローガンは、ゲームの本質を捉え、記憶に残るものでなければなりません。セールスポイントは、そのゲームの独自性と魅力を際立たせるものでなければなりません。これは、コアとなるアイデアから外側に広げ、次にそれを最もインパクトのある形に絞り込むプロセスです。まさに、真に革新する余地がある、この種の挑戦こそが、私が本当に楽しむのです。
