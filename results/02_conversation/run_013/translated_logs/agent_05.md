田中と渡辺はいる。どちらも僕と同じように直感的で、少し若い。田中は39歳、とてもエネルギッシュだ。渡辺は35歳、こちらも直感的だ。どちらか一人から始めるのが本当に良いと思う。気楽にチャットするだけで、プレッシャーも特定の議題もなしに、どんなアイデアが湧いてくるか見てみよう。もしかしたら、三人とも？直感的な小さなグループで、まるで熱いジャガイモを投げ合っているようにアイデアを出し合おう。それが僕が動き出すための最も効果的な方法だと思う。お互いの未熟な考えを積み重ね、興味深い方向へ追っていくことができる。
