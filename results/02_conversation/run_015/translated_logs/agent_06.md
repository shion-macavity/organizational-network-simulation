全員を部屋に集め、ホワイトボードと軽食を用意して、自由に話し合うことが、最初の最良の手順だと考えている。創造性が育つ環境を作り、誰もが自分の最も奇抜な考えを共有することに快適に感じられるようにする必要がある。そこから、洗練させ、私たちの集団的な想像力を本当に刺激するものを見つけ始めることができる。
