    *   **独自のセールスポイント（USP）：** 何が私たちのゲームを際立たせるのか？まだ誰もやったことがないこと、または私たちがより良くできることは何か？
    *   **ターゲットオーディエンス:** 誰のためにこのゲームを作っているのか？自由なアイデアは良いことですが、漠然としたプレイヤーを念頭に置くことで、創造的なエネルギーに集中できる場合があります。
    *   **潜在的なストーリーフック/世界:** ナラティブのためのアイデアの種に過ぎなくても、コンテンツを推進することができます。
3.  **具体性を促す（ただし、早すぎない）：** いくつかの大まかな方向性が得られたら、それらのアイデアがゲームメカニクス、潜在的なストーリーライン、ユニークな機能にどのように翻訳されるかについて、詳細を促します。たとえば、誰かが「歴史的なゲーム」を提案した場合、「どの時代ですか？どの特定の歴史的な出来事や人物ですか？どのようなひねりを加えることができますか？外交、戦闘、探検、またはまったく別のことですか？」完全に非歴史的なアイデアの場合、幻想的な要素、SFのトロープ、または抽象的な課題について聞きたいです。
4.  **キャッチコピーとセールスポイント:** これらは、しっかりとしたコンセプトが得られた後に自然に生まれてきます。チームとして、キャッチーでインパクトのあるものになるように洗練させます。
