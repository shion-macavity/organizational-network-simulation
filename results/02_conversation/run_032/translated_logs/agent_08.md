# エージェント個人史(和訳): Kobayashi (ID: 8)

## ■ ターン 0
* **【初期思考】**
> なるほど、新しいゲームのコンセプトか。現代設定だけど、かなり自由にやらせてもらえるみたいだし、他の要素も取り入れられるって。それはいい、本当にいい。この年になると、ルールに固執しすぎると良いアイデアが潰れてしまうことを学ぶんだ。直感的、それが俺のスタイル、いつもそうだった。物事を肌で感じて、火花が散るのを見て、それがどこに着地するかを見るのが好きだ。

さて、誰に話しかけるのが先かな？というか、どうやってこのボールを転がそうか？俺の最初の直感はいつも、ただ考え始めること、つまりアイデアをじっくりと煮詰めることだ。でも、それを洗練させるには、違う視点が必要になる。

山口は部署のリーダーで、彼も直感的だ。若くて29歳だが、直感的だからこそ、素っ気なく、口から飛び出すようなアイデアを理解してくれるかもしれない。俺が彼に突拍子もない考えをいくつかぶつけても、彼は眉一つ動かさず、むしろ気に入るかもしれない。彼がこの内容を大きなプレゼンテーションでまとめるんだから、早い段階で彼の賛同を得るか、少なくとも彼の最初の反応を聞くのは賢明だ。彼が簡単に却下するようなものを何日もかけて洗練させるのは無駄だ。

次に中村がいる。彼も直感的で、俺より少し若くて37歳だが、それでも彼の方が年上だ。ブレインストーミングに関しては、波長が合うかもしれない。直感的な頭が二つある方が、壁にスパゲッティをぶつけるよりも良い結果になることが多い。気軽にアイデアをぶつけたり、深く考えすぎずに初期の相乗効果を生み出すのに適した相手だろう。

さて、 KatoとYamadaは、どちらも熟考型だ。Katoはかなり若くて26歳、Yamadaは49歳だ。彼らは、ある程度形になったものを叩き台にしてから、「どうして」「なぜ」という質問をするタイプだ。それは俺にとって価値のあることだ、本当にそう思う。俺のアイデアは時々、少し…突飛なところがあるから、彼らはそれを地に足の着いたものにし、実現可能にしてくれる。しかし、反応できるような、具体的な何かをまず手に入れる必要があると思う。そうしないと、最初から分析麻痺に陥ってしまうかもしれない。それは俺のスタイルには合わない。

それで、俺の最初の計画は？今日は、頭を自由に彷徨わせることに時間を使うつもりだ。フィルターはかけない。新鮮な感じの現代設定とは何か？単なるシューターや街づくりゲームではない。ひねりを加えた何かだ。例えば、日常に隠された静かな謎、あるいは馴染みのある場所でのユニークな社会 dynamics のようなものだ。プレイヤーが人間関係、結果、そして日常生活におけるちょっとした予期せぬ出来事を探索できるようなものを考えている。人生シミュレーションゲーム、だが、強い物語性と、プレイヤーの選択が大きな影響を与え、分岐する結果をもたらすようなものだ。サンドボックスではなく、解き明かされる物語だ。

いくつかのコンセプトが浮かんできたら、おそらく明日朝に、山口にすぐに話しかけるだろう。「新しいプロジェクトについて考えているんだけど、いくつか突拍子もないアイデアがあるんだ。聞いてもらえないか？」という感じで。彼の興味を引くものがあるかどうか見てみる。そして、もし彼が暇なら、中村にも、よりカジュアルなブレインストーミングのために声をかけてみる。生のエネルギーを生み出すんだ。

他の部署については、まだ心配する必要はない。俺たちのリーダーは彼らのリーダーと繋がっているので、共有したり、相互に影響を与えたりする必要がある場合は、そのチャネルがある。しかし、今は、自分たちのチーム内で本当に面白いコアアイデアを生み出すことに集中している。まず、しっかりとした基盤が必要だ。

そうだ、それが計画だ。まず直感的に考え、次に他のメンバーを巻き込んで磨き上げていく。いいね。
