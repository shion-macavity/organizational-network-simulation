# エージェント個人史(和訳): Nakamura (ID: 7)

## ■ ターン 0
* **【初期思考】**
> さて、これを整理してみよう。新しいプロジェクト、自由なアイデア、物語、コンセプト、コンテンツ、スローガン、主要なセールスポイント…そして軍事/戦争というジャンルはあくまで提案。それは実際にかなり解放されている。思索的な人間として、私の最初の直感は、頭に浮かんだ最初のアイデアをすぐに叫ぶことではない。少し処理する必要がある、じっくりと浸透させる必要がある。

さて、部門のジャンルは軍事/戦争。ほとんどの人はすぐにシューターやリアルタイムストラテジーゲームに飛びつくだろうね、きっと。でも、ブリーフには「型にはまらない思考を歓迎する」と「他の要素を取り入れるか、そうしないかはお任せする」とある。これは面白い。もし、軍事や戦争の*本質*を取り、全く異なるものに適用したらどうだろう？あるいは、あまり一般的な側面には焦点を当てない、という方法も考えられる。

私は人間の要素について考えてみるかな。単なる戦闘ではなく、その*周り*で起こること、あるいは*後*で起こること。あるいは、それを防ぐこと。もし「戦争」が単なる戦いではなく、イデオロギー、資源、あるいは単なる個人的な葛藤といった対立のことならどうだろう？

よし、アプローチを考えてみよう。私はただ一人でブレインストーミングを始めることができる。それが私の自然な傾向だ。いくつかのコアアイデアを書き出し、それから少し洗練させる。しかし、これはチームの努力であり、部門には私と同じように思索的な Kato や Yoshida がいるし、リーダーの Yamaguchi さえも、直感的ではある。そして、別の部門の Sports 部の Sato がいる。もし頼めば、全く新しい、偏りのない視点を与えてくれるかもしれない。

私の最初の目標は、最終的なアイデアを*提示*することではなく、アイデアを*生成*することだ。だから、まずは自己反省をして、初期の、フィルタリングされていない考えをいくつか書き出してみようと思う。直接的な戦闘を伴わない「軍事/戦争」とは、どのようなものだろうか？あるいは、「軍事/戦争」とライフシミュレーションやパズルを組み合わせたものは？

例えば、戦争を*防ぐ*ために資源とロジスティクスを管理するゲーム。あるいは、紛争地域で非戦闘員として生き残り、コミュニティを築き、倫理的な選択をするライフシム。あるいは、敵の通信を解読したり、脱出経路を構築したりするパズルゲームのような、より抽象的なもの。可能性は本当に無限だ。

「軍事/戦争」をテーマにしたゲームの、あまり型にはまらないアプローチについて少し考えてみることにする。古いコンセプトを再利用したくない。いくつかのアイデアの種を見つけたら、実際に話せるようなものになったら、誰に相談するのがベストか考えるだろう。おそらく、同じように思索的な Kato や Yoshida に最初に相談して、協力してアイデアを構築できるかもしれない。あるいは、もし私が自分の頭の中で考えすぎていると感じたら、Kobayashi やリーダーの Yamaguchi のような直感的な人が、より自発的な考えで私を打破してくれるかもしれない。もちろん、別の部門の Sato もいる…スポーツゲームのプランナーが軍事テーマにどのように取り組むかは、本当に予想外になるかもしれない。しかし、今のところは、自分の頭の中でいくつかの初期コンセプトを形作ることだけに集中する。
