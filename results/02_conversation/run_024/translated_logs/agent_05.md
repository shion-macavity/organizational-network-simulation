## ■ ターン 0
* **【初期思考】**
> 新しいプロジェクトですね、これは。 「ストーリー、コンセプト、内容、スローガン、そして主要なセールスポイントを、自由な発想に基づいて設計する」。 「固定観念にとらわれないように」。 楽しそう！ 私の得意な仕事だ。

さて、誰に相談して、どのように進めるべきだろうか？

私の仕事のスタイルは直感的だから、まず最初に誰かに相談するのではなく、自分の頭でじっくり考えることにしたい。 アイデアの閃きや、大まかなコンセプトが浮かんだら、それを少し膨らませてから、周りの人に意見を聞くのが私のやり方だ。「自由な発想」や「固定観念にとらわれないように」という指示は、まさに私の脳の働き方に合っている。 もし最初から誰かと話してしまうと、特に内省的な高橋さんや田中さんのような方々と話すと、少し制約を感じたり、彼らの分析的なアプローチが、私の最初の創造性を阻害してしまうかもしれない。 まずは何かを形にしたい。

だから、「このタスクへのアプローチ方法」としては、まず私自身だけで、純粋でフィルターのかかっていないブレインストーミングセッションを行うことになる。 少しの間、「スポーツゲーム企画部」という部署名を無視するつもりだ。なぜなら、他の要素を取り入れたり、たとえ希望すればジャンルを無視することもできると言われたからだ。それは素晴らしい。 あらゆるゲームプレイのスタイル——アクション、RPG、ライフシム、パズル、何でもしっくりくるものがあれば——を自由に思いつく限りのアイデアを書き出す。 ストーリー、コアメカニズム、ユニークな設定など、どんな突飛なアイデアでも書き留める。 新鮮でエキサイティングで、人々の注目を集めるようなものを作りたい。 プレイヤーが「わあ、こんなの見たことない！」と思わせるようなものを作らなければならない。

いくつかの強力な、未加工のコンセプト、あるいは少なくとも私がワクワクするような本当にしっかりとしたコンセプトをいくつか手に入れたら、*その時*、誰に相談するかを考え始める。

「最初に誰に相談するか」についてだが、最初のアイデアを得た後、おそらく高橋さんと話したい。 彼も一般メンバーだが、内省的で、私の直感的なスタイルとは良い対比になる。 彼の方が私より10歳年上なので、より経験があるが、部署のリーダーではない。 彼なら、私の熱意の中で見落としたかもしれない潜在的な落とし穴を指摘したり、コンセプトの論理を完全に破壊することなく、それを洗練させるのに役立ってくれるだろう。 ゲームのメカニズムやストーリー要素の「どのように」と「なぜ」について、私が最初に見落とす可能性のある方法で考えてくれるかもしれない。 彼が「素晴らしい！」と言うだけではなく、また、考えずに突飛なアイデアをすぐに却下することもないだろう。

山本さん、私たちのリーダーは最終的に要約して提出する人だから、最終決定の前に必ず彼と話す必要がある。しかし、私は彼に、初期の考えの寄せ集めではなく、より洗練されたコンセプトを持ち込みたい。 彼に早めに話しかけると、まるで彼に私の初期のブレインストーミングを手伝ってもらっているように感じてしまうかもしれない。それは彼の役割ではない。

要するに：
1.  **アプローチ方法:** まずは私自身だけで、純粋で自由な形式のブレインストーミングを行う。 ストーリー、コンセプト、メカニズム、スローガン、セールスポイントの初期の、突飛なアイデアを書き出す。 創造性を抑えるな。
2.  **最初に相談する人（初期のブレインストーミング後）:** 高橋さん。 彼は内省的で経験豊富なので、私の直感的なアイデアを完全に潰すことなく、洗練させるための良い相談相手になるだろう。
