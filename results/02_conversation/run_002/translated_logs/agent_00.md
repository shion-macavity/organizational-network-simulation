## ■ ターン 0
* **【初期思考】**
> よし、新しいプロジェクトだ！ゲームをデザインしよう。自由なアイデア、型破りな発想、どんなジャンルでも構わない。私の部署は「スーパーヒーローゲーム企画」だけど、あくまでもスタート地点だ。これは面白そうだ、どんどんアイデアを練り上げていくのが大好きだ。

まず、「スーパーヒーロー」はベースだけど、ただの殴り合いにしたくない。どうにか捻りを加えられないか？もし、本当の試練が巨大ロボットと戦うことではなく、日常生活を送りながら秘密の正体を守ることだったらどうだろう？例えば、街を救う必要がある一方で、猫にご飯をあげるために家に帰る時間も守らないと、最低賃金で働く仕事に遅刻してしまう。ありふれた問題を抱えるスーパーヒーロー、ヒーローアクションとライフシミュレーションを組み合わせたもの。新鮮だ。

もしかしたら、「スーパー」とは、単なる力ではなく、ヒーローであると同時に普通の生活を送るために払う並外れた努力のことなのかもしれない。例えば、評判を管理したり、ソーシャルメディアでの反発に対処したり、あるいはデートを試みたりすること。ゲームプレイのスタイルとして「ライフシミュレーション」が言及されたが、それによってスーパーヒーローゲームを本当にユニークにできると思う。

だから、私のアプローチとしては、まず自分の考えを自由に彷徨わせ、ストーリー、ヒーローの種類、普通の生活がどんなものか、そしてどんなユニークな課題に直面するかについて、たくさんの突飛なアイデアを書き出すことから始めるつもりだ。まだ何も検閲しない。とにかく全部書き出そう。

そして、いくつかの強力で奇妙なコンセプトができたら、誰かに相談したい。誰か、直感的に考える人も必要だ。…ああ、高橋だ！彼女も直感的だ。まず彼女と話すのが良いと思う。お互いに刺激し合い、境界線を押し広げ、本当に型破りなものを生み出すことができるだろう。詳細にこだわらず、コアとなるアイデアを固め、エキサイティングにするために、まずは初期の、未完成の考えを共有したい。後で洗練させることはできるが、今は純粋で、何もないブレインストーミングに集中する。

ヒーローであることの戦闘以外の側面、人間的な要素、秘密の正体の葛藤をぜひ探求したい。もし、彼らの力があまり目立たない、あるいは不便なものであればどうだろう？例えば、あなたのスーパーパワーが超スピードであるにもかかわらず、信じられないほどのスピードでカロリーを消費し、常に空腹であるとか。あるいは、飛べるが、乗り物酔いになる。それによって、奇妙なコンテンツの層が加わる。

そうだ、まずは自分の頭の中でいくつかの非常に大まかなコンセプトをスケッチし、次に高橋に会って、どんなクレイジーなアイデアを一緒に生み出せるかタイミングを見計らおう。これは良いものになるぞ。
