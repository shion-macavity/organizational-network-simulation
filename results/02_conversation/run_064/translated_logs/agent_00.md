では、この課題にどう取り組むか？
まず、一人で時間を過ごしてブレインストーミングをする。私が興味を持つさまざまな歴史的時代やテーマを書き出し、それからねじ曲げてみる。歴史的なものをどのようにして、完全に予想外のゲームプレイのスタイルや、幻想的/SFの要素を与えることができるだろうか？例えば、古代帝国という概念を取り上げて、プレイヤーがさまざまなタイムラインをまたぎ、歴史的パラドックスを防ぐために協力する協力型パズルゲームと組み合わせるのはどうだろう？あるいは、歴史的時代を舞台にしたライフシミュレーションだが、手続き的に生成された、少し時代錯誤の世界で、強いクラフトと探索の要素があるもの。主要なセールスポイントは、そのユニークなブレンドから生まれるだろう。

別の部署… 幻想ゲーム企画の中村さん。それは興味深い選択肢だ。もし私が、ファンタジーを強く傾けているコンセプトや、それを歴史と非常にユニークな方法でブレンドするコンセプトを思いついた場合、中村さんと話すことは本当に有益だろう。彼らは、ファンタジー要素を魅力的にするとは何か、またはそれを付け加えられたように感じることなく、どのようにスムーズに統合できるかについての洞察を持っているかもしれない。また、私たちの「歴史的」なコンセプトが、私たちの部署の通常の焦点から外れた場合にどのように認識されるかを知ることもできる。しかし、まず自分の部署に焦点を当てて、コアアイデアを具体化してから、広範囲にネットワークを広げるつもりだ。
