## ■ ターン 0
* **【初期思考】**
> 新しいゲームプロジェクト、ですか。そして、完全に型破りな発想をしてほしいと。この年齢で、まさにそういう挑戦が好きです。細かい部分にすぐ入り込むのではなく、まずは壮大でワクワクする全体像を捉えることに重点を置きたい。私の仕事ぶりを「直感的」と呼ぶ人もいますが、まさにそうありたいと思っています。まだ厳密な枠組みは必要ありません。

部門は「ポストアポカリプス」ですが、他の要素やゲームスタイルについては自由に手を出すことができるとのこと。素晴らしいですね。ただの埃っぽい荒野を舞台にしたシューターにしたくありません。そういうのはもうたくさん見ました。ポストアポカリプスの世界を、単なるサバイバル以上のものとして、何が*面白い*のかを考えています。再建はどうでしょうか？瓦礫の中に美を見つけるのは？あるいは、天知らずですが、ちょっとした予想外の喜びはどうでしょうか？

まず最初に、いくつか生のアイデアを出さなければなりません。リーダーの山本さんに、未熟な考えをすぐに持ち込むつもりはありません。彼女も私と同じように直感的ですから、良いコンセプトを評価してくれるでしょう。しかし、彼女に話す前に、もう少し具体的な内容を用意しておきたいと思っています。同じことが、伊藤さんにも言えます。彼はもっと「バランス型」なので、この初期段階では、私が持っているよりも少し構造が必要になるでしょう。

いいえ、この初期のブレインストーミングには、直感と突飛なアイデアを好む人、すぐにコンセプトを投げかけても、すぐに詳細を気にしなくても大丈夫な人が必要です。直感的な流れを「理解」してくれる人。

佐藤さんは51歳、私も直感的です。私と同じように、たくさんのゲームコンセプトを見てきたのでしょう。あまりフィルターなしで、アイデアを出し合えるはずです。とにかく、何でも壁にぶつけて、何がくっつくか見てみましょう。あるいは、鈴木さんがいます。彼は34歳と若く、私も直感的です。私がまだ考えてもいない、まったく異なる新鮮な視点を持っているかもしれません。そこから何が起こるか見てみるのも面白いでしょう。

まずは佐藤さんと話をしてみようと思います。年齢も仕事のスタイルも近いので、これらの初期段階の非常に抽象的なアイデアについて、すぐに共通の波長を見つけられる可能性が高いでしょう。ジャンルではなく、「どんな*体験*をプレイヤーにさせたいのか」、感情や雰囲気について話しましょう。お茶かコーヒーを飲みながら、気軽に話して、どんな奇妙で素晴らしいコンセプトが湧き上がってくるか見てみましょう。

では、どのようにアプローチしますか？おそらく、佐藤さんの机に歩み寄ったり、休憩室で彼女を見かけたりするでしょう。例えば、「佐藤さん、この新しいプロジェクトについて、何か突飛な考えはありますか？自由にやらせてもらっているんですが、私の頭の中は、ただゾンビを撃つだけではない、色々なことでいっぱいです」と言って、そのまま会話を進めます。最初の焦点は、ゲームの*雰囲気*と*核となるコンセプト*に置き、まだ具体的なゲームメカニズムにこだわらないようにします。何をするのか、なぜするのかが分かってから、「どのように」を心配すればいいのです。

ポストアポカリプスというジャンルをひっくり返すようなものを考えています。もし、世界の終わりを生き残ることではなく、新しい始まりで繁栄することだったらどうでしょうか？あるいは、荒廃の中に美を見つけることだったら？戦わずに、つながりを築いたり、自然を回復させたり、忘れられた物語を発見したりするゲームはどうでしょうか？本当に予想外のことです。そういう「型破り」な発想を、私はテーブルに持ち込みたいと思っています。
