## ■ ターン 0
* **【初期思考】**
> よし、集中していこう。田中、52歳、直感的。さて、全く新しいゲームのアイデアについて話しているんだ。完全に自由形式で、SFにこだわらなくてもいい、というのも実は結構ワクワクする。ストーリー、コンセプト、コンテンツ、キャッチコピー、主要なセールスポイント…考えるべきことは多い。そして、固定観念にとらわれない発想を求められている。気に入っている。本当に気に入っている。

まず最初に、いくつかの初期的な考えを書き出す必要がある。直感的な人間として、私はあまり初期的な構造を設けずにアイデアを自由に流していくことで本当に活力を感じる。まずは落書きをしたり、頭に浮かんだキーワードや非常に抽象的な概念をメモしたりして始めようと思う。判断は一切せず、純粋なアウトプットに集中する。ありきたりなものだけでなく、本当にワイルドなゲームプレイのスタイルを探求したい。例えば、「プレイヤー」が従来のキャラクターではない、あるいは目標が表向きのそれとは異なるようなもの。そういう「固定観念にとらわれない」発想に惹かれるんだ。

「このタスクにどう取り組むか」について言うと、最初のステップは間違いなく一人でのブレインストーミングセッションだ。私、ノート、そして美味しいお茶があればいい。心を自由に彷徨させて、何が引っかかるか見てみよう。この段階では、実現可能性や一貫性も気にしない。興味を引くものであれば、何でも数ページ埋め尽くすようにする。例えば、「もしライフシミュレーションと…深宇宙探査を組み合わせたら？」とか、「パズルゲームが会話なしで説得力のあるミステリーを語れるだろうか？」といったことを考えてみる。少なくとも3つの明確で高レベルのコンセプトを、たとえそれが単なるアウトラインでしかなくても、思いつくようにする。

いくつかの非常に粗く、未完成のアイデアができたら、誰に相談するかを決める必要がある。私の直感的な仕事のスタイルを考えると、初期の生のアイデアを、同じように直感的に考える誰かにぶつけ合うのが次のステップとして最適だと思う。すぐに構造化したり批判したりしようとしない人、むしろ創造的なエネルギーを付け加える人だ。佐藤が思い浮かぶ。彼は私も直感的だ。抽象的な考えをさらに投げ込み、芽生えたアイデアを拡張したり、もし私のアイデアが少し古くなっているなら、完全に新しいアイデアを呼び起こしたりするのに適任だと思う。コーヒーでも飲みながら、気軽に話して、プレッシャーなしにアイデアを自由に流し合おう。まだ「決定的な」アイデアに落ち着く必要はない。一緒に創造的な風景を探求することだ。彼なら私が気づかないつながりを見つけたり、私が行き詰まっていることに対して新しい視点をもたらしたりするかもしれない。

より思索的な Watanabe や、バランスの取れた Ito や Yamamoto に相談するのは、佐藤と私が創造的な力を最大限に発揮し、さらにワイルドな考えをいくつか組み合わせて、少しばかりまとまりのあるものに（それでも非常に粗いながらも）できる *後* にするのが良いと思う。今は、できるだけ多くの創造的な可能性を生み出すことが重要だ。他の部署…それは面白い考えだが、もう少し形になったものが、本当に外部の視点から恩恵を受けることができるようになったら、後回しにしておこう。まずは自分の部署内で初期的なフィードバックを得た後、別のチームの誰かに完全に焼ききっていない考えをぶつけるのは避けたい。
