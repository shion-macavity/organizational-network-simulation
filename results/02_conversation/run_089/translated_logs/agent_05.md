さて、この初期段階の突飛なアイデアを相談するのに最適なのは誰だろう？ リストを見てみたところ… 高橋さんは思慮深いから、後で洗練させるにはぴったりだけど、初期の「何でもあり」の段階には向かないかもしれない。田中さんはバランスが取れているのは良いけど、本当に突飛なアイデアを出すには、僕と同じように直感的な思考をする人が必要だ。

そうだ！ 渡辺さんだ！ 彼も直感的だ。完璧だ！ 僕よりは経験豊富だ、58歳だからね。でも、直感的であれば、きっと「何でも壁にぶつけて何がくっつくか」というアプローチも理解してくれるだろう。きっと僕らと相性が良くて、本当にユニークなものを一緒に生み出せるはずだ。僕の型破りなアイデアも、彼なら評価してくれると思う。

だから、戦略はこうだ：

まず、少し時間をかけて、頭の中で自由にさまよい、浮かんでくるどんなランダムなコンセプトやストーリーのフックでも書き留める。フィルタリングも、判断もなし。純粋なブレインストーミングだ。コーヒーでも飲んで静かな場所を探すか、あるいはただ歩き回るかもしれない。
