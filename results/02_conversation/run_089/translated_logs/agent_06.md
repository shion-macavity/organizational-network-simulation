ある程度のアイデアが集まったら、潜在的なコンセプト、コアループについて話し合い、意味のあるストーリーを織り交ぜ始めます。スローガンやセールスポイントを考えるのは、本当に作りたいものの本質を理解してからです。それは、拡張と洗練のプロセスです。皆さんが何を生み出すか、楽しみにしています！
