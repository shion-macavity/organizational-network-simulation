## ■ ターン 0
* **【初期思考】**
> よし、新しいプロジェクトだ！これはワクワクするな、アイデアを自由にぶちまけることができる。 「自由なアイデア」「固定観念にとらわれない思考」、まさに私の得意分野だ。現代設定だけど、他の要素も入れられるのかな？最高だ。そんな自由が大好きだ。

さて、目標は物語、コンセプト、内容、スローガン、主要なセールスポイントだ。やるべきことは多いけど、それがまさに私が好きなことだ—ゼロから何かを構築する、その火花から始める。

まず最初に、生のアイデアをどんどん出していく必要がある。私の仕事のスタイルは直感的だから、普段はすぐ飛び込んで、コンセプトが湧き上がってくるのを待つ。最初は、実現可能性をあまり気にせず、とんでもない可能性を頭の中でスケッチする時間を少し費やすだろう。純粋な想像力だ。何がクールだろう？これまでまったく同じやり方でやったことはないだろうか？思いもよらない方法でジャンルを融合させたもの、例えば、エリート秘密工作員でもあるライフシム、あるいは、予想外の深い感情的なストーリーが展開されるパズルゲームなど。そういうひねりが大好きだ。

いくつかの強力で、もしかしたらクレイジーなコンセプトが頭の中で渦巻いたら、誰に相談するかを考え始める必要がある。特に私の直感的なスタイルでは、最初のブレインストーミングには、あまり深く考えずにアイデアをどんどん出せる人と話したいと思う。

リストを見て…渡辺さん！彼も直感的だ。完璧だ。お互いにアイデアを出し合い、素早く構築することで、本当に良い相乗効果を生み出せるだろう。時には、頭の中にある狂ったコンセプトをさらに狂わせたり、自分が考慮していなかったクールな角度を見つけ出すために、誰かの助けが必要なこともある。だから、最初の内部ブレインストーミングが終わった後、まず彼に話を聞いてもらいたいと思う。コーヒーでも飲みながら、「ねえ、突拍子もない考えがあるんだけど、聞いて、どう思う？」みたいな感じで。

私のアプローチは、まずいくつかの明確なコアコンセプトを生成することだ。それぞれのコンセプトについて、私はすぐに以下をスケッチしようと思う。
1. **フック:** 最初の「わあ」となる要素とは？
2. **コアループ:** プレイヤーは主に何を *する* のだろうか？
3. **感情的な核:** どんな感情を呼び起こしたいのだろうか？

それから、これらのスケッチを渡辺さんに持って行き、一緒に詳細に詰め、どのコンセプトに最も可能性がありそうか議論し、さらには異なるアイデアの要素を組み合わせることができる。このプロセス全体は、未加工のダイヤモンドを見つけることだ。コアコンセプトが完全に確定的になるまで、詳細なストーリーや具体的なゲームメカニクスを書くのは無意味だ。それが私が通常、最高の成果を得る方法だ—直感的な爆発、そして共同での洗練。
