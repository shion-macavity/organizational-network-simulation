## ■ ターン 0
* **【初期思考】**
> さて、完全に新しいゲームのアイデアを考えて出さなければならない。ストーリー、コンセプト、コンテンツ、キャッチコピー、セールスポイントなど、すべてがアイデアの自由である。ジャンルは「ファンタジー」だが、非常に柔軟で、ゲームプレイはなんでも構わない。これは、実はおもしろいな。本当に想像力を自由に羽ばたかせられる、この種のオープンエンドな課題は大好きだ。私の「直感的」な仕事のスタイルは、この種の環境でこそ本当に活きる。

まず最初に、どうアプローチするか。直感的な人間なので、一人で部屋に座って、空白の壁をじっと見つめ、アイデアを無理やり引き出そうとするのはあまり好きではない。私の最高のアイデアは、物事を話し合ったり、誰かと考えをぶつけ合ったり、あるいはただ自分の心をさまよって、一見無関係なものを結びつけたりするときに思いつくことが多い。だから、まずは大量の未加工のアイデアを生成することから始めたい。まだ自己検閲は不要、とにかくすべて書き出そう。

私にとって良い出発点となるのは、私にとって「ファンタジー」とはどういう意味なのかを考えること、そしてそれをどのように裏切ったり、拡張したりするかを考えることだ。エルフやドワーフだけでなく、未来的な設定のファンタジーはどうだろう？あるいは、「魔法」が実際には忘れられた技術であるファンタジーはどうだろう？あるいは、夢全体でできたファンタジーの世界はどうだろう？また、プレイヤーにどのような*感情*を与えたいかを考慮したい。畏敬の念、神秘、挑戦、安らぎ、驚き？その感情は、しばしばゲームプレイのスタイルを決定する。私は、探検と発見を促し、プレイヤーの選択が本当に影響力を持つような、強い物語要素があるものを好む傾向にある。あるいは、魔法のペット養護施設を経営するファンタジーライフシムのような、まったく型破りなものかもしれない！可能性は無限だ。

さて、誰に最初に相談するか。私の仕事のスタイルを考えると、間違いなく直感的な人にもう一人話したい。彼らは、わがままなアイデアにオープンであり、すぐに「非現実的」または「ニッチすぎる」という理由でコンセプトを閉ざす可能性が低いだろう。この段階では、コンセプトを構築することが重要であり、それを打ち砕くことではない。

連携する同僚リストを見ると、小林と吉田がすぐに目につく。彼らも私と同じように「直感的」な仕事スタイルとしてリストされている。
小林は37歳で、私より少し若いため、私が考慮しないかもしれない、本当に新鮮な視点やアイデアを持っているかもしれない。彼女のエネルギーは、多くの初期コンセプトを生成するのに最適だろう。
吉田は52歳で、私より年上だが、彼も直感的だ。彼は、おそらく豊富な経験と、最もクレイジーなアイデアでさえ、その可能性を見抜く才能を持っている。彼は、より抽象的な直感的なアイデアを、より具体的で、しかし創造的なものに落とし込むのにも役立つかもしれない。

まずは小林とカジュアルにおしゃべりすることにしようと思う。彼女は年齢が近く、私たち両方とも一般メンバーなので、気軽にブレインストーミングをするのに適した方法だと感じる。どんなにばかげたものであっても、思いつくものを何でも思いつくままに投げかけ、何がうまくいくか見てみよう。大量の未加工のアイデアを生成した後で、吉田を迎えたり、あるいは「バランス型」の Kato を迎えたりするかもしれない。Kato は、アイデアを洗練させ、それが実際に実現可能かどうかを確認するのに役立つだろう。しかし、今は純粋なアイデア出しが目標だ。

そう、私の計画はこうだ。小林とのプレッシャーのかからないブレインストーミングセッションから始めて、直感的な心を自由に、可能性でいっぱいにする。ファンタジーのさまざまなひねり、さまざまなゲームプレイのスタイル、そしてどのようなユニークなプレイヤーエクスペリエンスを作成したいかを探索する。次に、アイデアが煮詰まってきたら、それらを選別し始める。
