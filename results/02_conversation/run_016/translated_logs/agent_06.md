私の最初の目標は、全員を活性化させ、純粋に創造的に考えさせることだ。「歴史」は我々の部署の名前であり、このプロジェクトのアイデアに対する鎖ではないことを理解してもらう必要がある。本当にユニークでエキサイティングなものを作り上げよう！
