私の仕事のスタイルは内省的で、つまり、最初に思いついたことをすぐに口に出すタイプではありません。アイデアをしばらくの間、じっくりと味わわせ、頭の中で反芻し、さまざまな角度から見てみたいのです。ある程度の準備なしに議論を始めてしまうのは、基礎を築かずに家を建てようとするようなものです。ですから、私にとって最初のステップは、当然、考えることです。本当に、真剣に考えるのです。どんなゲームコンセプトなら、本当にワクワクするだろうか？どんな体験を創造する手助けをしたいだろうか？

「スポーツ」の部署というラベルを考えると、スポーツのコンセプトを全く新しいものに仕立て上げようとするのが良いかもしれません。例えば、ボクシングの試合のように激しい、競争型の料理ゲームはどうでしょうか？あるいは、主人公の主な目標が究極の庭師になることである、ライバルのガーデニングクラブや壮大な植物バトルが展開されるライフシミュレーションはどうでしょうか？それは「スポーツ」としては明らかに型破りです。あるいは、全く関係のないものかもしれません。アンティークショップを舞台にしたミステリーゲーム、あるいは、深海探検家であるプレイヤーたちが協力してパズルを解くゲームはどうでしょうか？可能性はかなり広いです。

まず、いくつかのコアアイデアを思いつく必要があります。ストーリーのフック、ユニークなゲームプレイメカニズム、プレイヤーに体験してもらいたい感情です。いくつかの漠然としたアウトラインができたら、誰に相談すべきかを検討し始めます。

小林さんは良い選択肢です。彼は私の経験と同様に、バランスの取れた仕事のスタイルを持っており、常に安定した手腕だと感じています。彼は、私の初期の、おそらく突飛なアイデアの、明らかな欠陥や現実的な課題を特定するのに役立ち、地に足の着いた視点を提供してくれるかもしれません。私は、答えを全て持っている必要がないと感じずに、半生の人に気軽に考えをぶつけることができるのです。

中村さんと佐々木さんは、彼らの直感的な仕事のスタイルで、後で非常に貴重な存在になるかもしれません。しっかりとしたコアコンセプトができたら、彼らは創造性の爆発を注入したり、エキサイティングな新しい要素を提案したり、その場でキャッチーなキャッチフレーズを思いつくのに最適です。しかし、初期の枠組みを確立するためには、より構造化された分析的なアプローチを好みます。小林さんのほうがそれに合っているかもしれません。

ですから、私のアプローチはこうです。
まず、私は独力でアイデアを練ることに真剣な時間を費やします。どんなにばかげたものであっても、すべて書き留めます。さまざまなジャンル、プレイヤーにとってのさまざまな感情的な体験、そしてユニークなコンセプトが市場で際立つ方法について考えます。ゲームの「なぜ」について考え、単に「何」について考えることを確認する必要があります。どんな感情を呼び起こすのか？その目的は何なのか？

いくつかの明確なコンセプトができたら、それぞれに大まかなストーリー、いくつかのコアメカニズム、そして潜在的なフックを伴って、最も強く、または2つのコンセプトを選びます。それから、小林さんに相談し始めると思います。彼に私のコンセプトを共有し、彼の率直な意見を求め、彼が私のアイデアを洗練させ、穴を見つけたり、さらに強化する方法を提案したりするのを手伝ってもらいます。彼は、自分の頭の中で深く考えているときに見落とすかもしれないことを、私に気づかせてくれる、バランスの取れた視点を持っています。それは、メインのリングに入る前に、予備の練習相手がいるようなものです。

小林さんとアイデアについてフィードバックを受け、洗練させた後、中村さんや佐々木さんの直感的な意見を取り入れるなど、他の人を関わらせることを検討できます。あるいは、より洗練されたコンセプトを山口さんに直接提示することもできます。しかし、今は私と私の考えだけです。仕事に取り掛かる時間です。
