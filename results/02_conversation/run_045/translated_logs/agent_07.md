さて、「ホラー」とは、現代において何を意味するのか？ ジャンプスケア？ 心理的な恐怖？ ボディホラー？ コスミックな実存主義？ 俺は常に、最も効果的なホラーは思考させ、現実や自身の道徳を問い詰めるものだと感じている。見ることだけでなく、見ないこと、あるいは見ることのできるものこそが重要なんだ。

キャッチコピー… 「楽園の種は、あなたの悪夢の根を隠している」。あるいは、「未来を築く… 恐ろしい秘密を一つずつ」。

主要なセールスポイントは以下の通り：
1. **ジャンルのユニークな融合**: ライフシミュレーション/協力型建築と、心理的/宇宙的ホラーの欺瞞的な融合。

4. **より深い謎を解き明かす**: プレイヤーは、自分たちの環境とその住人の真の性質を解き明かす。
