まず最初に、皆を集めなければならない。私のやり方は、本当に自由な発想を促すブレインストーミングセッションを始めることだ。最初からあまり主導せず、舞台設定をするだけに留めたい。会社が何を求めているのかを明確に伝える。ストーリー、コンセプト、コンテンツ、スローガン、主要なセールスポイント。すべてが「自由なアイデア」に基づいていることを伝える。そして、「終末世界」というジャンルは、私たちの部署の戦略的な方向性であり、檻ではないと明言する。RPG、パズル、さらには生活シミュレーションやレースゲームのような、他のジャンルの要素を自由に組み込んでもらいたい。この段階では、より珍しいアイデアほど良い。

重要なのは、単にゲームの「アイデア」を求めるのではなく、「感情」と「体験」を求めてほしいということだ。彼らはどのような世界を想像しているのか？どのような課題があるのか？どのような感情的な旅を体験させたいのか？荒涼とした世界にしたいのか、それとも活気に満ちた世界にしたいのか？孤独な世界にしたいのか、それとも共同体的な世界にしたいのか？まず、コアとなる体験に焦点を当てることで、そこから外側に構築していくことができる。

さて、誰に最初に相談すべきだろうか？正直に言うと、この最初のブレインストーミングでは、部署の全員に同時に相談するつもりだ。皆の生の、フィルターされていない思考を同時にテーブルに出したい。時には、ある人の少し変わったアイデアが、他の人の心に素晴らしい火花を散らすことがある。そのような相互作用を制限したくない。どんなアイデアも悪いアイデアではないという環境を作るようにする。特に、最初の方では。ただキャンバスにペンキを塗りたくっているだけだ。

今の私の直感としては、終末世界のいつもの陰鬱さを覆すようなものが良いのではないかと考えている。もしかしたら、サバイバルの要素が、ミュータントと戦ったり、廃墟と化した都市を襲ったりするだけでなく、美しさを再構築したり、喜びを見つけたり、あるいは終末後の世界で奇妙で素晴らしい新しい生命とコミュニティを発見したりすることにあるゲームだ。 「終末」が終わりではなく、むしろ根本的に新しい始まりであるとしたらどうだろう？プレイヤーが新しい生態系を育成したり、独自のポストヒューマン文化を育んだりするタスクを負う、協力型のライフシミュレーションやパズルアドベンチャーゲームなどだ。あるいは、異なる派閥が最も回復力のある、または審美的に美しい新しい世界を構築しようと競い合う、競争要素を取り入れることもできる。もちろん、これはあくまでも一瞬の考えであり、今は皆のアイデアを聞くために、自分の中で留めておくつもりだ。しかし、ジャンルを少しひっくり返すという方向性は、非常に魅力的だと感じている。

戦略：
1. 部署全体で会議を開催する。
2. 「自由なアイデア」の概要を明確に伝え、当初の部署の焦点を超えたジャンルやゲームプレイの制限はないことを強調する。
3. プレイヤーの体験や世界観について、幅広い思考を促すために、オープンエンドな質問から始める。
4. どんなに突飛なアイデアでも、最初はすべて記録する。
5. 初期ブレインストーミングの後、最も強力なコンセプトをグループ化、洗練、そして構築し始めることができる。その時こそ、田中と渡辺のバランスの取れた思慮深いアプローチが、物事を構造化する上で本当に活きてくるだろう。しかし、まずは純粋で、未加工の創造性を！これはとても楽しみだ。
