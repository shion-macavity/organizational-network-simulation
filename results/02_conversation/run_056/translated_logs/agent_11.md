# エージェント個人史(和訳): Yamada (ID: 11)

## ■ ターン 0
* **【初期思考】**
> さて、新しいゲームのコンセプトを練り始めるための準備をしよう。山田、45歳、直感的…よし、見てみよう。

まず最初に、どんなゲームを作ろうとしているのかを把握する必要がある。部門は「スライス・オブ・ライフ」だが、何でも追加できると言われたはずだよね？そして「型破り」がモットー。それは良い、気に入った。もし革新を目指すなら、いつものやり方に固執する意味はない。

私の仕事のスタイルは直感的だから、頭の中でチェックリストを厳密にこなすのではなく、点と点が線で繋がったり、イメージが湧いてきたりする。通常、私が始めるのに最適な方法は、とにかく口に出して考えを整理したり、あまりにも深く考えずに初期のアイデアを書き出したりすることだ。ストーリー、コンセプト、コンテンツ、キャッチコピー、セールスポイント…これらをゼロから一人で取り組むのは大変だ。

では、誰に最初に話しかけるべきだろうか？佐々木さんだ。彼女も51歳で、私と同じ一般メンバーだ。彼女なら、遠慮なく突飛なアイデアを出し合えるだろうと確信している。最初は少し奇抜に見えるアイデアでも、互いの考えを発展させることができる。完璧な計画をすぐに固めることではなく、可能性の範囲を探求することだ。彼女は、そういった自由なアプローチを高く評価してくれると思う。お互いにぶつけ合い、何が残るかを見て、誰もが考えないような奇妙な角度を見つけるかもしれない。

もし、非常にバランスが取れていたり、方法論的な人から始めると、考えを早すぎる段階で構造化しなければならないと感じてしまい、それが本当に創造的な火花を消してしまうことがある。佐々木さんとなら、アイデアを自由に流れさせ、その後に、形が見え始めるかもしれない。

初期の、たとえ粗削りなコンセプトがいくつかできあがったら、その時で Yoshida-san や Kobayashi-san に話を聞くことを検討する。彼らはどちらも「バランス型」の仕事のスタイルなので、物事を洗練させたり、潜在的な問題を指摘したり、突飛なアイデアを実際にゲームとして *実現* する方法を見つけたりするのに役立つだろう。より幻想的な要素を地に足の着いたものにしたり、実用的なコンテンツのアイデアを提案したりしてくれるだろう。

そして、リーダーの山口さんも、直感的だ。佐々木さんと私が何かより確かなもの、あるいは少なくともいくつかの強力な方向性を持った後で、必ず話を聞きたい。彼は創造的なエネルギーを高く評価してくれるだろうが、それが実現可能な製品にどのように繋がっているかを見たいと思うだろう。リーダーに相談する際には、いくつかの強力な候補を持っていることが常に良い。

他の部門…うーん、最初に話しかけることに強い衝動はない。ともかくオプションだし。自分の部門の創造的なプロセスの中で確固たる基盤を築いてから、他の部門に手を伸ばす方が良いと思う。ここで十分な才能があるので、ユニークなものを生み出すことができる。特定の要素に壁にぶつかったり、新しい視点が必要になった場合に、後で話しかけるかもしれないが、最初のブレインストーミングには必要ない。

だから、計画はこうだ。
1. 頭の中を自由にさまよう。初期のコンセプトを内部でスケッチする。フィルタリングを気にせず、何が出てくるかを見る。
2. その後、佐々木さんに話しかける。例えば、「佐々木さん、新しいゲームの初期のアイデアを色々考えているんだけど、 ‘型破り’ の指示を受けて、とにかくアイデアを自由にぶつけ合いたいと思ったんです。コーヒーでも飲みながら、ストーリー、コンセプト、思いつくままに、本当に突飛なアイデアをブレインストーミングしませんか？」 そう言えば彼女の仕事のスタイルに響くと思う。いくつかの初期のストーリーフックや、予想外のゲームプレイの組み合わせを思いつくかもしれない。
3. そこから、チーム全体でより広く共有できるほど、まとまりのあるものに、浮かび上がってきたものを形作っていく。

うん、それで間違いない。どんなクレイジーなゲームのアイデアを生み出せるか見てみよう。心で、そして全く予想外のゲームを。それが課題だ。
