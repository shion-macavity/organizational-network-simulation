# エージェント個人史(和訳): Nakamura (ID: 7)

## ■ ターン 0
* **【初期思考】**
> さて、頭を動かしてみよう。新しいプロジェクト、新しいアイデア…いつも興味深い段階だ。ゲームのアイデアを、ストーリーからスローガンまで、ゼロから考え出す必要がある。ジャンルは「SFゲーム企画」だが、基本的に何でもできると言われた。それは良い、柔軟性を保てる。私は中村、50歳、色々な経験をしてきたし、物事をバランスの取れたやり方で進めるのが得意だと考えている。

さて、まず最初に「誰に相談すべきか」「どのようにこのタスクに取り組むべきか」だ。

「誰に相談すべきか」について、私の直感としては、この種のタスク、つまり「自由なアイデア」を生み出すことが求められている場合、まずは同僚との非公式なブレインストーミングから始めるのが良いだろう。いきなりリーダーの山口に相談すると、彼女がすでに特定のビジョンを持っている可能性があり、それが「自由なアイデア」の段階から制限してしまうかもしれない。まずは少し探ってみる必要がある。

チームを見回すと、小林が最初の相談相手として良さそうだ。46歳、こちらも一般メンバーで、私の「バランス」のようなワークスタイルだ。問題解決やアイデア構築に関して、私たちは似たような考え方をするだろう。構造的でありながら創造的なアプローチを高く評価するはずだ。お互いに初期のトップレベルのコンセプトをぶつけ合い、何が火花を散らすか見てみよう。たとえば、ユニークなSF設定や、既存のジャンルへのひねりを試してみるのも良い。どのようなコアなプレイヤー体験を創造したいかについて話すことができる。

加藤と吉田はどちらも「直感的」なタイプだが、それは斬新なアイデアを生み出すのに適している。しかし、最初のステップとしては、アイデアをある程度地に足の着いたものにするのを助けてくれる人が必要で、小林の方が適していると思う。小林と私がいくつかの初期の種を見つけたら、そこで加藤と吉田を巻き込んで、よりワイルドなアイデアを生み出すのが完璧だろう。そして、良いアイデアのプールができたら、その時に山口にプレゼンテーションを行い、戦略的な意見を求める。

他の部署については、直接的に連携リストにいる人はいない。オプションではあるし、後で新しい視点をもたらす可能性もあるが、この初期の「自由なアイデア」の段階では、まずは社内中心に集中するのが良いと思う。他のチームのビジョンとの相互作用や差別化を考える前に、まず「私たち自身の」ビジョンを定義する必要がある。それは、ゲストを招く前に自分の家を片付けるようなものだ。

さて、「どのようにこのタスクに取り組むべきか」？
私の「バランス」のワークスタイルとは、創造性と実用性の良いバランスを保つということだ。ただランダムな言葉を並べるだけではいけない。プロセスが必要だ。

1.  **初期ブレインストーミング（小林との）：**
    *   小林とカジュアルな会話を提案する。非常に広範囲から始めることができる。「プレイヤーにどのようなコアな感情を抱かせたいか？」「あまり探求されていないユニークなSFの前提条件は何か？」について話すことができる。さまざまなゲームプレイのスタイルについて話すこともできる。たとえば、伝統的なシングルプレイヤーのジャンルを協力型にし、またはその逆にするのはどうだろうか？ライフシムと深宇宙探査を組み合わせるのはどうだろうか？
    *   ここでは、「ストーリー、コンセプト、コンテンツ、スローガン、主要なセールスポイント」について考えている。おそらく、強力な中心となる*コンセプト*、つまり「フック」から始めて、その周りにストーリーとコンテンツを構築する。
    *   たとえば、思いつくままに言うと、何世代も旅をしてきた世代間宇宙船を舞台にしたゲームはどうだろう。その目的は忘れ去られ、乗組員は独自の孤立した文化と技術を発展させている。そして、ゲームは宇宙船の元の使命と目的、そして地球に何が起こったのかを発見することについて描かれる。

2.  **アイデアの拡張（加藤と吉田との）：**
    *   小林と私がいくつかの初期のコンセプトを見つけたら、そこで加藤と吉田を巻き込むことを提案する。彼らの直感的なスタイルは、ストーリー、ユニークなコンテンツのアイデア、そして世代間宇宙船のアイデア、または他に醸成されているコンセプトに対して、本当に驚くべきゲームプレイのメカニズムやひねりを肉付けするのに役立つだろう。彼らはキャッチーなスローガンを考えるのが得意だろう。

3.  **洗練と構造化：**
    *   ある程度の生のアイデアのプールができたら、それらを洗練する必要がある。ここが「バランス」の部分が本当に活きてくる場所だ。どのアイデアが最もまとまっているか？どれが最も可能性を秘めているか？主要なセールスポイントをどのように明確に定義できるか？ここがプレゼンテーション可能なものに形作っていく段階だ。
    *   世代間宇宙船のアイデアについて考えてみよう。宇宙船内のさまざまな派閥、忘れられた過去の謎、内部の対立や外部の脅威の危険性について考えることができる。「コンテンツ」には、資源の管理、さまざまなセクターの探査、古代の宇宙船システムに関連するパズルの解決、または外交や戦闘の遭遇が含まれる可能性がある。

4.  **山口へのプレゼンテーション：**
    *   最後に、ストーリーのピッチ、コアコンセプト、潜在的なコンテンツ、スローガン、主要なセールスポイントを完全に備えた、いくつかの十分に開発されたアイデアができたら、山口にプレゼンテーションを行い、フィードバックを得て、次に進むアイデアを最終決定する。

だから、私の戦略は、協調的でピアに焦点を当てて始める、アイデアを構築する、創造的な入力を拡大する、そして最後に洗練して提示することだ。そう思う。
