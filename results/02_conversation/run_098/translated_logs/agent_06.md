1. **まず広い範囲で、次に焦点を絞る:** 可能な限り広い範囲でスタートしたいと考えています。 間違ったアイデアや、ばかげたコンセプトはありません。 「型破りな」考え方をすることを明示的に指示されているので、まさにそれを奨励します。「スーパーヒーロー」というジャンルは、始まりに過ぎず、檻ではありません。 ヒーローではなく、生き延びようとするサイドキック、またはカフェを経営する悪役のスーパーヒーローゲームにするかもしれません。 あるいは、スーパーヒーローゲームにライフシミュレーションとミステリー要素を組み合わせたものにするかもしれません。 何でもありです！
2. **反復的なブレインストーミング:** 大きく、オープンな議論から始め、コンセプトを思いつくままにどんどん出していきます。 次に、いくつかの有望なアイデアを選び、もう少し深く掘り下げてみます。 大まかなストーリーをスケッチし、独自のゲームプレイメカニズムについて考え、プレイヤーエクスペリエンスがどのようなものになるかを本当にイメージしてみます。
3. **コアとなる柱を定義する:** 方向にいくつかの強力なものが見つかったら、一緒にコアコンセプト、主なストーリーの展開、ゲームを埋めるコンテンツの種類、キャッチーなスローガン、そして魅力的なセールスポイントを定義します。 これらは明確でエキサイティングなものであり、私たちを本当に差別化するものにしたいと考えています。

私は間違いなく初期のブレインストーミングを主導し、皆が安心してアイデアを共有できるようにします。 物事を始めるために、自分自身でホワイトボードにいくつかの初期のフレームワークをスケッチするかもしれませんが、その後はチームにそれを埋めてもらうようにします。 私の仕事は今、促進し、刺激することです。 皆さんの最高の力を引き出し、本当に境界線を押し広げられるようにすることです。 これはとても楽しみになるでしょう！ 何ができるかを見るのが楽しみです。
