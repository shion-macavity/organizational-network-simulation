## ■ ターン 0
* **【初期思考】**
> さて、新しいゲームのストーリー、コンセプト、内容、スローガン、そして主要なセールスポイントを考えていくことになりました。そして「自由なアイデア」「型破りな発想」…これは結構ワクワクしますね！ 私たちの部署は「スーパーヒーローゲーム企画」ですが、大まかな方向性なので、ジャンルを自由に組み合わせられます。アクション、RPG、パズル、ライフシム…選択肢がたくさんあります。

まず、いくつかの初期アイデアを書き出す必要があります。 「自由なアイデア」なので、最初からあまり制限しない方がいいでしょう。 スーパーヒーローゲームで、悪者を殴り倒すだけではないものは？ あるいは、それを新鮮な視点からどう捉えるか？

考えているのは… 確立されたスーパーヒーロー *自身* になるゲームではなく、スーパーヒーローになる *過程*、あるいは失敗して自分の道を見つけるゲームはどうだろうか？ あるいは、スーパーヒーローである *傍ら* の日常を管理するゲームはどうだろうか？ 家賃をどう払うのか？ 自分の正体をどう隠すのか？ ライフシミュレーションとスーパーヒーローの要素を組み合わせると、とても面白くなるかもしれません。

まずは、コアコンセプトをブレインストーミングしましょう。 「Heroic Hustle: Civilian Secrets（ヒーローいぶき：市民の秘密）」はどうでしょうか？

**コンセプトアイデア:** あなたはスーパーヒーローではなく、生活を立て直し、普通の生活を維持しながら、密かに力を持っているか、あるいは地下のスーパーヒーローネットワークの一員である、普通の人間として生活するライフシミュレーションRPGです。 コアゲームプレイループは、仕事、社交生活、そして力やヒーローとしての義務をバランスさせることです。

**ストーリーアイデア:** あなたは、賑やかな半未来都市に住む、学生ローンや安定した仕事を探している若い大人です。 ある日、あなたはユニークな力を偶発的に発現させます。最初は、どんな空間でも完璧に整理できる、人々の気分を微妙に操作できる、あるいは単に信じられないほどの幸運など、一見すると些細な力かもしれませんが、可能性を秘めています。あなたは、新興力を抱える人々を支援し、二重生活を送るのを手伝う、秘密の非公式「ヒーロー育成機関」にスカウトされます。 メインストーリーは、普通の市民と隠れたヒーローコミュニティの両方を脅かす都市規模の陰謀を明らかにし、自分の正体を隠しながら、好奇心旺盛な大家、同僚、そして出会いの相手から身を隠しながら、創造的な方法で力を使い続けることを中心に展開します。

**内容（ゲームプレイ要素）:**
*   **昼/夜サイクルとスケジュール:** 仕事（例：バリスタ、プログラマー、デリバリードライバー）、社交（友人、家族、デート）、そして秘密のヒーロー訓練/ミッションの間で、エネルギー、お金、時間を管理します。
*   **力の進化:** あなたの初期の力は進化します。 「完璧な整理整頓」なら、重力を利用して掃除したり、精密な修理のために念力を使ったりするかもしれません。 「微妙な影響」なら、本格的な感情操作や念読みに発展するかもしれません。 ミッションや日常生活での選択が、あなたの力の進化の方向性を形作ります。
*   **スキルツリー:** 力以外にも、「市民スキル」（例：より良い仕事の見込みを増やすためのカリスマ性、移動のための運動能力、ハッキングのためのテクノロジーの知識）と「ヒーロースキル」（例：ステルス、戦闘、推理）を発展させます。
*   **人間関係システム:** NPCとの友情、ライバル関係、またはロマンチックな関係を築きます。 これらの関係は、新しいクエストのロックを解除したり、サポートを提供したり、あるいはあなたの秘密を暴露したりする可能性があります。 一部のNPCは、他の秘密のヒーローやヴィランかもしれません！
*   **倫理的な選択:** 多くのミッションには、道徳的なジレンマが含まれます。 猫を助けるために力を使い、身元を暴露するリスクを冒しますか？ より大きな利益のために、汚職した役員を助けますか？ あなたの選択は、ヒーローとしての評判と市民としての評判に影響を与えます。
*   **カスタマイズ:** キャラクターの外見、アパートの装飾、さらには「秘密基地」（単に改造されたクローゼットかもしれません）をカスタマイズします。

**スローガン:** 「あなたの街。あなたの秘密。あなたの物語。」 あるいは「普通のヒーロー。非凡な人生。」 後者の方が、バランスを強調しているので、少し気に入っています。

**主要なセールスポイント:**
1.  **ユニークな「スライス・オブ・ライフ」スーパーヒーロー体験:** 秘密のアイデンティティの関連性の高い苦労に焦点を当て、日常の課題と並外れた力をブレンドします。
2.  **深いプレイヤーの自由度:** 意味のある選択が、あなたの個人的な生活とヒーローとしての旅の両方に影響を与え、複数のストーリーパスとエンディングにつながります。
3.  **創造的な力の進化:** ユニークで非伝統的な力から始めて、あなたのプレイスタイルに基づいて成長し、適応させます。
4.  **豊かでダイナミックな都市:** 多様なキャラクター、隠された秘密、そして偶発的なイベントで満たされた活気のある都市環境。

さて、これはかなりしっかりとした基盤だと思います。 スーパーヒーローの要素がありながら、それをライフシミュレーションRPGに転換させているので、伝統的な「スーパーヒーロー」ゲームとしては間違いなく型破りです。
