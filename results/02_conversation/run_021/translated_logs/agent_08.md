# エージェント個人史(和訳): Kobayashi (ID: 8)

## ■ ターン 0
* **【初期思考】**
> 新しいプロジェクトのブリーフだ。「デザインストーリー、コンセプト、内容、スローガン、主要なセールスポイント。」そして一番重要なこと：「自由なアイデア」「固定観念にとらわれない発想」「ジャンルはあくまで広範な戦略的方向性」。まさに自分の得意分野だ！ついに、創造性を存分に発揮できるチャンスが来た。直感的な部分がすでに様々な可能性で活気づいている。

さて、誰に相談するのが最初で、どのようにアプローチすべきだろうか？

まず最初に必要なのは、生の情報だ。白紙の状態では、誰にも相談できない。自分の仕事のスタイルは直感的だから、普段はアイデアを自由に流れ出させて、フィルターや判断を一切入れずに、しばらく時間をかけることが多い。おそらく、ノートやホワイトボードの前に座って、思いつくままに書き出し、まずは「ミリタリー/ウォーゲーム」の部分を完全に無視するだろう。どんな感情を呼び起こしたいのか？どんな体験をプレイヤーにしてもらいたいのか？何か馴染みのあるものに、本当に新鮮で予想外のひねりを加えられるか？それとも、全く新しいものなのか？

いくつかの初期の火花、いくつかの粗いコンセプトができた後で、誰に相談するかを考え始めることができる。

リーダーの山口さんは、私と同じように直感的だ。彼はワイルドなアイデアに興奮するだろう、それは素晴らしいことだが、最終的に概要を提出するのは彼だ。だから、彼に相談する前に、アイデアを少し洗練させて、本当に考え抜いた結果だという印象を与えたい。最初の生半可な考えをそのまま受け取られて、さらに発展させる機会を失いたくない。

中村さんは思慮深い。彼は豊富な経験を持っている。アイデアの弱点を見抜いたり、潜在的な問題を指摘したり、より堅牢で実用的なものにする方法を提案するのに適している。完全に突飛で未熟なコンセプトを最初に彼に相談するのは避けるだろう、なぜなら彼はもっと構造を求めてくるだろうから。より具体的なコンセプトができ、実現可能性について現実的なチェックや具体的なアドバイスが必要になったときに彼に相談する。技術的なことではなく、コンセプト自体の確固たるものについてだ。

加藤さんと山田さんは、どちらも思慮深く、私の年齢に近い。どちらか一人が、気軽に話をするための最初の窓口になるだろう。たぶん、加藤さんからだ、彼女は私の同い年だから、より気軽に話せるかもしれない。いくつかの非常に粗いアイデアを彼女に話して、どんな反応があるか見てみるだけだ。彼女が私が見落としていることに気づいてくれるかもしれないし、彼女の思慮深さが、私が「かっこいい」要素だけに集中しているために見過ごしていた別の角度を見せてくれるかもしれない。彼女は、まだ漠然としているかもしれないが、コアコンセプトをもう少し明確にするのに役立ってくれるかもしれない。承認を得るためではなく、より具体的なものを構築する前に、新鮮で思慮深い視点を得るためだ。

計画はこうだ。まず、一人でじっくり時間をかけて、自由連想法とブレインストーミングを行う。本当に奇抜でエキサイティングなアイデアをいくつか書き出す。それから、加藤さんにコーヒーでも飲みながら、「この新しいプロジェクトのために、本当にワイルドなアイデアがあるんだけど、私が完全に頭がおかしいかどうか教えてくれない？」と話してみるつもりだ。そうすることで、様子を探り、初期のフィードバックを得て、正式に何かを提示する前に、物事を形作っていくことができる。
