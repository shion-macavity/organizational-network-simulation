では、この課題にどのように取り組むか？
まず、一人でじっくりと考える時間を作るだろう。最初に「日常系」の本質について考える。「日常系」とは、一体何を意味するのか？ 日常の経験、人間関係、ささやかな喜び、時間の流れ、コミュニティ意識や個人的な成長などだ。しかし、それをどのように捻じ曲げるか？ 「生活」が予想外のものであるとしたら？ 「切り口」がはるかに大きく、幻想的なパイから切り取られたものだとしたら？

プレイヤーにどのような感情を呼び起こしたいのかを考えるだろう。居心地が良くリラックスした気持ちにさせたいのか？ ワクワクして冒険心に駆り立てたいのか？ 興味をそそり好奇心を刺激したいのか？ その中心となる感情体験が、物語とコンセプトを導くことになる。

次に、対照的なアイデアをいくつか検討する。例えば、「日常系」ゲームは通常、現実的だ。もし宇宙だったら？ あるいは魔法の世界だったら？ あるいはキャラクターが人間ではなく、意思を持つ物体、あるいは精霊だったら？ そして、彼らの「生活」のどのような「切り口」が面白いだろうか？ 忘れられた神社の守護精霊として、時折迷い人を迎えたり、季節の移り変わりに対応したりする生活の「切り口」かもしれない。あるいは、賑やかな未来都市で小さな、役に立つロボットとして生活する「切り口」かもしれない。

これらの幅広いコンセプトをいくつか頭の中で渦巻かせ、非常に基本的なストーリーのフックやゲームプレイのループを考えた後、佐藤さんに連絡を取るだろう。これらのアイデアが、設定が完全に異なっていても、「日常系」の「心」を保っているかどうか話し合うことができる。内省的で、しばしば個人的な旅にふさわしい、深みや独自のメカニズムの可能性を探ることができる。そして、プレイヤーに馴染みがあり、かつ斬新な体験を提供できるかどうかを議論する。

スローガンやセールスポイントをすぐに決定することはない。それは、コアコンセプトとストーリーが形になり始めてからだ。今は、広い網を投げて、最も有望なアイデアをゆっくりと、慎重に引き上げることに集中する。プレイヤーに考えさせ、世界と少しでもっとつながっていると感じさせる、本当に新鮮で魅力的なものを生み出すことを目指している。どんな世界であれ。
