## ■ ターン 0
* **【初期思考】**
> さて、頭をフル回転させよう。課題は、コンセプトからスローガン、セールスポイントまでゲームのアイデアを考え出すことだ。想像力を存分に働かせる。ジャンルに制限はないのが素晴らしい。部署の一般的な方向性は「日常系」だが、そこから派生させたり、完全に無視したりすることもできる。少し羽を伸ばす良い機会だ。

まず最初に、初期のアイデアをいくつか書き出す必要がある。誰に話しかけるか考える前に、思いつく限りのことを、どんなに突飛なものでも、とりあえず書き留めておこう。自由連想法だ、というやつだ。

その初期のブレインストーミングの後、いくつかのコンセプトがまとまってきたら、誰かと話したいと思う。話し合いの相手として最初に思い浮かぶのは、渡辺さんだ。彼女は私と同じ年齢で、同じく一般メンバーで、仕事のスタイルも私と似ている。波長が合いそうだし、遠慮なくアイデアを出し合えると思う。特に、未熟なアイデアでも、同じような考え方をする相手と発展させやすい。遠慮なく「もし～だったら」というシナリオを深く掘り下げ、すぐに洗練させる必要がない中で、さまざまな方向性を探求できる。それに、彼女も同じ部署なので、私たちの一般的な役割に対する共通認識がある。境界線を押し広げているとはいえ。

アプローチとしては、少し意外性はあるけれど、それでも共感できるようなものから始めるだろう。例えば、最初は日常系として始まるが、やがてはるかに壮大で幻想的なものへと展開していくようなものだ。例えば、完璧な日常を送っているのに、突然…すべてを見直すことを強いるような、非日常的な出来事が起こる、というようなものだ。そこからアドベンチャーゲームになったり、ミステリーになったり、あるいは、皆で協力して世界の新しいルールを見つけ出すようなゲームになったりするかもしれない。

おそらく、2、3個の非常にラフなコンセプトを書き留めた後で、渡辺さんに声をかけるだろう。「渡辺さん、少しお時間ありますか？新しいプロジェクトのアイデアを色々考えているんですが、かなり突飛な内容なんです。もし何か引っかかるものがあれば、あるいは、あなた自身が全く斬新なアイデアを持っていれば、聞いてみたいと思ったんです。2つ頭の方が1つ頭より良いでしょうから、特に、型破りな発想を求められているときはね。」

それが良い出発点だと思う。まずは頭の中を空っぽにして、何かを洗練させる前に、同僚と協力してアイデア出しをする。そうすれば、プロジェクトのブリーフにある「自由なアイデア」の側面を最大限に活用できる。
