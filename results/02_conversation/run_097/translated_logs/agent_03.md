## ■ ターン 0
* **【初期思考】**
> さて、どうなることか…。新しいゲームプロジェクト、ゼロから、自由なアイデア、部門名である「スーパーヒーローゲーム企画部」を超えたジャンルに制限はない（それはあくまで提案だし）。ストーリー、コンセプト、内容、スローガン、主要なセールスポイント…。思いつくのは大変だ！そして、リーダーの山本さんが最終的にすべてをまとめることになる。よし、田中、深呼吸だ。

まず最初に、自分の考えを整理する必要がある。「型にはまらない発想を」と言ったな。もし、スーパーヒーローのコンセプトを少しひねってみたらどうだろう？いつものように、強力なヒーローが世界を救うのではなく、ヒーローの*周囲のコミュニティ*や、ヴィランと戦っていないときに彼らが直面する*日常の苦労*について、もっと焦点を当てたらどうだろう？あるいは、ヒーローが伝統的な意味でのヒーローではないとしたらどうだろう？

ふむ、僕は23歳で、斬新なアイデアが好きだが、実際にゲームにできるものが何かもわかっている。バランスの取れた働き方をするので、創造的でありながら、ある程度現実的な何かを考えるべきだろう。

だから、ヒーローがスーパーヴィランと戦うのではなく、ヒーローの「超能力」がもっと微妙で、もっと日常的で、しかし深い影響を与えるものだったらどうだろう？たとえば、ゲームはカフェの経営についてだが、主人公や客が、本当に微妙で、ほとんど平凡な「超能力」を持っていて、興味深い課題や機会を生み出す、といった具合だ。誰かが完璧なコーヒーを淹れることができるが、ストレスを感じているときだけ、とか、誰かがそばにいるだけで誰でも落ち着かせるが、ひどく内気な人、といった具合だ。

いや、それはあまりにもライフシミュレーションに傾きすぎていて、「スーパーヒーロー」という広義の定義からも外れてしまう。少しでもその英雄的な雰囲気を保ちながら、ひっくり返してみよう。

よし、プレイヤーが伝統的なヒーローではなく、新進のヒーローのための*サポートシステム*だったらどうだろう？ヒーローの*オリジンストーリー*に焦点を当てたゲームだが、彼らが新しい力と責任を乗り越えるのを助ける人の視点から見たものだ。大惨事と日常の問題の両方に対処することを含む。

ああ、面白い！少しライフシミュレーションと、物語冒険、そしておそらく戦略的な要素が混ざったものだ。これを「サイドキックの物語」とか「ヒーローの旅：サポートスタッフ版」と呼ぼう。

**コアコンセプト:** あなたは、超能力を発見したばかりの誰かの個人的なアシスタント、ハンドラー、あるいは親しい友人/家族としてプレイします。あなた自身が力を得るのではなく、ヒーローの生活を*管理*することです。秘密と公の生活の両方です。

**ストーリー:** 主人公は、新しい仕事として個人的なアシスタントになるが、すぐに彼の上司が実は新米のスーパーヒーロー、「キャプテン・カタリスト」であることを知る。彼は意図せず、奇妙なラボ事故で力を得た。キャプテン・カタリストは不器用で、心優しいが、完全に圧倒されている。物語は、プレイヤーがキャプテン・カタリストが普通の生活（請求書の支払い、力誘発性の不安に対するセラピー、能力の制御方法の学習）と、芽生えつつある英雄的な義務（軽犯罪者の阻止、付随的な損害への対処、公人になることの学習）のバランスを取るのを手伝うことを中心に展開する。スライス・オブ・ライフの瞬間、道徳的ジレンマ、そして最終的に、キャプテン・カタリストがより多くの名声を獲得するにつれて現れるより大きな脅威がある。

**内容/ゲームプレイ:**
*   **毎日の管理:** 時間管理要素。プレイヤーはキャプテン・カタリストのスケジュールを組む：能力制御のためのトレーニングセッション（ミニゲーム）、公の場への登場、普通の人のための予定、そして犯罪との戦いのミッション。
*   **問題解決:** 対話の選択肢とミニパズルで、キャプテン・カタリストが社会的な状況、倫理的なジレンマ（例：「汚職政治家を暴露するか、銀行強盗に焦点を当てるか？）、個人的な問題を乗り越えるのを手伝う。
*   **リソース管理:** 彼らの公のイメージ、減り続ける貯金（英雄行為はお金を稼がないから！）、そして精神的な健康を管理する。新しい「ヒーローのスーツ」の素材を買うか、家賃を払うかを選ぶ必要があるかもしれない。
*   **人間関係の構築:** 警察の皮肉屋の刑事、親切な科学者、しつこい記者、キャプテン・カタリストの心配そうな家族など、他のキャラクターとの関係を築く。あなたの選択はこれらの関係に影響を与え、異なるストーリーパスをアンロックする。
*   **「非戦闘」の英雄的行為:** 直接的な戦闘に焦点を当てない。代わりに、戦略を立て、情報を収集し、キャプテン・カタリストの特定の、しばしばぎこちない力を創造的に使用して、過度の破壊や暴力なしに状況を解決する。キャプテン・カタリストの力は固体を変えてゼリー状にするものだとすると、誰かを傷つけずに強盗を止める方法を考える必要がある。

**スローガン:** 「すべてのヒーローの後ろには、伝説がある。」あるいは、「世界を救うという本当の仕事は、スポットライトが当たる*前*に始まる。」

**主要なセールスポイント:**
1.  **ユニークな視点:** 完全に新しい、人間的な視点からスーパーヒーローのジャンルを体験する - 混沌を管理する無名のヒーロー。
2.  **物語の深さ:** プレイヤーの選択、道徳的ジレンマ、そして並外れた力と普通の生活のバランスを取るという、関連性の高い苦労によって推進される豊かな物語。
3.  **魅力的なゲームループ:** 時間管理、パズル解決、人間関係構築の魅力的な組み合わせで、プレイヤーは常に自分の役割に夢中になる。
4.  **心とユーモア:** 痛ましい瞬間と軽快なユーモアのブレンドで、地上からスーパーヒーローの生活の魅力と不条理を示す。
5.  **戦略に焦点を当て、戦闘に焦点を当てない:** 純粋なアクションではなく、戦略的思考と物語重視のゲームを愛するプレイヤーにアピールし、英雄性に対する思慮深いアプローチを提供する。

よし、これはしっかりとした基盤だ。スーパーヒーローのテーマを受け入れながら、真にユニークな方法でそれをひねっている。感情的な深さと遊び心のあるシナリオの両方に潜在力がある。
