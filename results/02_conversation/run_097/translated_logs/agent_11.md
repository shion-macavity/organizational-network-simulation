## ■ ターン 0
* **【初期思考】**
> さて、新しいプロジェクト、自由なアイデア、これはワクワクする一方で、正直に言うと少し気が重い。 「スポーツゲーム企画」という部署のタグは、まるで提案のようであり、ルールという感じではないのが、なかなか面白い。 彼らは本当に私たちに型破りな考え方をしてもらいたいようだ。 まったく白紙だね。

まず最初に、私が作りたいゲームの種類、少なくとも探求したいゲームの種類を把握する必要がある。 私の仕事のスタイルはバランス型なので、最初の考察なしに突っ走るようなことはしたくない。しかし、意見を聞く前に考えすぎてしまうのも避けたい。

ジャンルが単なる広い戦略的な方向性であることから考えると、スポーツの要素を取り入れることができるが、厳密には伝統的なスポーツシムではないようなものを考えている。 スポーツの競争心を全く異なる文脈で使うゲーム、あるいはプレイヤーの旅が何らかのスポーツイベントや成果に結びつくようなライフシミュレーションゲームだが、それが最初から*コア*なゲームプレイではない、というのもありかもしれない。 あるいは、大規模なスポーツイベントの舞台裏で起こるミステリーゲームで、スキャンダルなどを暴く、というのはどうだろう？ それは興味深いかもしれない。

私は、物語とプレイヤーの選択に焦点を当てたもの、RPGや強いストーリー要素を持つライフシムのようなものが良いのではないかと考えている。 そこでの「スポーツ」の要素は、単に試合に勝つだけでなく、キャラクターの成長、目標の達成、あるいは個人的な課題の克服といったものに重点が置かれることになる。 もし、例えば、協調パズルゲームとスポーツテーマを融合させ、プレイヤーが大きなゲームのための「トレーニング」や「戦略」として提示される課題を解決するために協力しなければならないが、伝統的な意味でのゲームをプレイすることはない、というゲームを作ったら、本当にユニークになるかもしれない。 あるいは、リズムゲームで、「ビート」がアスレチックな動きと同期される、というのもありかもしれない。 ちょっと突飛かもしれないが、彼らは「自由なアイデア」と言ったんだ！

よし、いくつか漠然とした方向性が浮かんできた。 自分の手でフルストーリーやコンセプトを練り上げる前に、それはすぐにウサギ穴に落ちてしまう可能性があるから、誰かに最初の、生のエッセンスをぶつけてみるのが賢明だと思う。 早期のフィードバックを得て、何が響くかを見て、そして私の仮定に異議を唱えてもらう必要がある。

誰が最初の会話に適任だろうか？ 佐々木さんが思い浮かぶ。 彼は私のくらいの年齢で、思慮深いところも気に入っている。 遠慮なく、壁にスパゲッティをぶつけてみて、何がくっつくかを確認できる、良いやり取りができるだろう。 彼は私たちの部署にいるので、より広い文脈にも詳しい。しかし、彼も広い視野を持っていることは知っている。 彼に相談すれば、おそらく「未来的なスポーツリーグを舞台にしたミステリーゲームはどうだろうか？」とか「キャラクターはアスリートだが、ゲームではなく個人的な成長と人間関係に焦点を当てたライフシムはどうだろうか？」といった、いくつかの非常に大まかなコンセプトを提示するだろう。 彼の反応を見て、何か初期の火花が散るかどうかを確かめたい。

小林さんや吉田さんを検討することもできる。 彼らは年長で思慮深いので、より経験に基づいた視点を提供してくれるかもしれない。 潜在的な落とし穴や、自分が考慮していなかった強力な方向性を見つけてくれるかもしれない。 しかし、この最初の、非公式なブレインストーミングセッションでは、佐々木さんが気軽にわいわいとアイデアを交換するのに最も適した人物のように思える。 私たちはどちらも一般メンバーなので、リーダーである山口さんにいきなり相談するよりもプレッシャーが少ない。 リーダーに話す前に、少し洗練された方向性を持っておきたい。

だから、今の私のアプローチは、頭の中にいくつかの非常に異質で型破りなゲームコンセプトを集め、それから佐々木さんに非公式なブレインストーミングを依頼することだ。 そうすることで、初期の反応を得て、自分の手で深く掘り下げる前に、より具体的なものを形作ることができる。 どんな「自由なアイデア」が本当に脚光を浴びるかを絞り込むのに役立つだろう。
