## ■ ターン 0
* **【初期思考】**
> よし、集中していこう。新しい製品、物語、コンセプト、コンテンツ、スローガン、セールスポイント… 全てを考え出す必要がある。しかも、軍事/ウォーゲームの計画とはいえ、「型にはまらない」発想をとのこと。これは結構面白いな、ジャンルやゲームプレイに厳密な制限もない。直感的な部分がすでに様々な可能性で活気づいている。

さて、まず「誰に相談すべきか」。僕は直感的タイプだから、アイデアを出し合ったり、誰かの刺激からクリエイティブな火花をsparkさせたり、ただ話し合ってどこへ向かうかを探ったりするのが好きだ。まだ未熟なアイデアをいきなりリーダーの山本さんに持って行っても、最善とは言えないだろう。彼はバランス型だから、もう少し構造化されたものを評価するだろうし、今の僕のアイデアはただの生の可能性の寄せ集めだ。

同僚を見ても… 佐藤はバランス型で、後で実用的なアドバイスをもらいやすいだろう。伊藤は思索的で、深い思考や批判には最適だが、どんなアイデアも悪いアイデアではないという初期の自由なブレインストーミングには向かないかもしれない。

でも、鈴木ならどうだろう！彼女も直感的だ。完璧じゃないか！お互いにアイデアを出し合い、判断することなく、最もクレイジーなコンセプトを投げ合える。2つの直感的な頭脳が協力すれば、まさに求められている境界線を押し広げることができる。部門のテーマと全く予想外のものを組み合わせた、本当にユニークなものを生み出せるかもしれない。彼女なら、複雑なパズルで戦うウォーゲーム、退役軍人がコミュニティを再建するライフシム、あるいは魔法のメカが登場するファンタジー世界を舞台にした軍事戦略ゲームのような、かなり突飛なアイデアにもオープンだろう。すぐに現実的である必要がない限り、可能性は無限だ。

だから、まずは鈴木に、昔ながらのブレインストーミングセッションを持ちかけるのが確実だろう。

そして、「このタスクにどう取り組むか」についてだが、鈴木との会話の前や会話中に、ただ心を自由に彷徨わせるつもりだ。まずは「軍事/ウォー」という部分を無視するか、少なくとも完全に捻じ曲げることから始める。兵士や戦車を超えて、「戦争」とは何か？アイデアの戦争か？自然との戦争か？自己との戦争か？外交や貿易で戦う戦争か？そして、それをゲームとしてどのようにして楽しく魅力的にできるか？

様々なアートスタイル、感情的なトーンについて考える。心温まる戦争物語にできるか？恐ろしいものにできるか？面白いものにできるか？
「自由なアイデア」と「型にはまらない」という指示を最大限に活かしたい。戦争の終結に焦点を当て、再建と和解を描く戦略的要素のあるゲームはどうだろう。あるいは、紛争が純粋に知的である「戦争」、軍事テーマの派閥を持つ競争的なパズルゲームはどうだろう。あるいは、歴史的な戦いを音楽でリードする行進バンドを率いるリズムゲームのように、音楽で士気を維持するようなものも。

そして、鈴木と一緒になって、これらの突飛なアイデアを壁に投げつけ、何がくっつくかを見て、より有望なものをいくつか、物語、コンセプト、コンテンツ、キャッチーなスローガン、そしてセールスポイントを具体化できるほどに洗練させよう。何を作り上げるにしても、単なるクローンではなく、本当に際立って見えるものにしなければならない。それが課題であり、正直なところ、一番楽しい部分だ。
