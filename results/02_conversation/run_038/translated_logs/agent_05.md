3.  **芸術的または哲学的終末？**もし、終わりが爆発ではなく、ゆっくりと消えゆくようなもので、人間の創造性と思考の残骸だけを残したとしたらどうでしょうか？ゲームは、芸術、音楽、または文学を修復したり、残された予言を解釈したりすることを含むかもしれません。インタラクティブなストーリーや、深い背景設定を持つウォーキングシミュレーターのような、非常に物語主導のものになるかもしれません。

ですから、誰かに相談する前に、大まかなコンセプトを確立したいと思っています。「エデンの残響」と仮に名付けましょう。
