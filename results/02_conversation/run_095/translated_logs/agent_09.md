## ■ ターン 0
* **【初期思考】**
> 新しいゲームプロジェクトだ！これは楽しみだ。 「自由なアイデア」「固定観念にとらわれない思考」「どんなゲームプレイ形式でも」… それは耳に心地よい響きだ。SFゲーム企画部の一員として、自然と星、高度な技術、異星の世界に思考が向かうが、今回の指示では自由に組み合わせても良いとのことなので、最高だ。 私はそういう自由を求めているんだ。

さて、どう取り組むか？私の仕事のスタイルは直感的で、まずひらめきが最初に浮かび、その後で逆算したり、点と点を結んだりする傾向がある。 ゼロから構造化されたブレインストーミングを無理やり行うのはあまり好きじゃない。 何かきっかけが必要なんだ。

私がこのプロジェクトに取り組むのに最適な方法は、自分の思考を少しさまようさせて、何がクリックするか見てみることだと思う。 恐らく、ジャンルや特定のメカニズムではなく、プレイヤーに体験してもらいたい本当にクールな *感覚* やユニークな *瞬間* について考えることから始めるだろう。 例えば、巨大で生きた異星のエコシステムをナビゲートする感覚を本当に感じられるとしたらどうだろうか？ あるいは、一見すると見渡す限り何もない惑星で孤独な探検家だが、あなたが触れるものすべてに隠された歴史があるとしたらどうだろうか？

いくつかの抽象的な「雰囲気」のアイデアを手に入れたら、実際のゲーム要素を重ねていくことができる。 その感覚に合うような物語はどんなものだろうか？ コアメカニズムは何だろうか？ 古代の異星言語を解読するパズルゲームなのか、厳しい世界を協力してテラフォームしようとする協力型サバイバルゲームなのか？ 世代船で新しい社会を築くライフシムなのか？ 可能性は無限大で、早すぎる段階でドアを閉めたくない。

相談についてだが… 最初にアイデアを相談するのに最適なのは、中村さんに話を聞くことだと感じている。 彼も直感的な仕事のスタイルを持っており、経験も豊富だ。 彼もきっとたくさんの奇抜なアイデアを持っているだろうし、私のアイデアが少し突飛でも批判しないだろう。 彼は未完成のコンセプトを拾い上げて、さらに発展させたり、自分のアイデアをさらに良くするような意外な要素を投げ込んだりするのに適任だろう。 彼から「正解」を得るというよりは、私たち双方にとってさらに多くのアイデアを刺激するような活発な会話を持つことが重要だ。 まるで一緒にキャンバスにペンキを塗り、何がくっつき、どんな新しい形が現れるかを見るようなものだ。

まずは、本当に生々しく、フィルターのかかっていないアイデアを頭の中に湧き上がらせると思う。 自己検閲はしない。 それから、今朝のうちに軽く中村さんの席に立ち寄り、お時間があれば少しお話を聞かせてもらおうと思う。「中村さん、少しよろしいでしょうか？ 新しいプロジェクトについて考えていて、変なアイデアが次々と浮かんできたので、どう思うか聞きたかったんです」 そうして始めるのが、私にとって最良の方法であり、直感的なアプローチを最大限に活用できると思う。
