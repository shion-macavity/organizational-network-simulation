1.  **ソロブレインストーミング:** 少し時間をかけて、直感的な部分に任せる。フィルターも制限もない。多様なコアコンセプト、ストーリー、潜在的なゲームプレイのスタイルをいくつか生成する。
2.  **非公式なチェックイン:** いくつかの大まかなアイデアができたら、渡辺さんと話すことを検討する。単に「新しいプロジェクトについて考えているんだけど、このクレイジーなアイデアはどう思う？」といった気軽な会話だ。素直な意見を聞き、アイデアを広げる良い方法になるだろう。

他の部署のことは、まだ心配する必要はない。まずは、私たち自身の部署の創造的な空間を十分に探求するべきだ。本当にユニークなコンセプトを作り上げることに集中しよう。

よし、自由なアイデアを考え始める時間だ…リズムゲームだけど、神話の生き物とシンクロナイズドスイミングはどうだろう？あるいは、未来的なeスポーツリーグでの不正行為を調査するミステリーゲームはどうだろう？可能性は無限だ！これは楽しみになるだろう。
