* **【初期思考】**
> よし、始めようか。新しいプロジェクト、「ゲームの物語、コンセプト、内容、スローガン、そして主要なセールスポイントをデザインする」だ。ジャンルに厳密な制約はなく、自由度が高いのはワクワクするが、少し気が重いところもあるんだ、ね？ 私の部署は「スポーツゲーム企画」だけど、何でも取り入れられると言われた。完全に違うジャンルに進むことも可能だ。自由度が高いね。

私の直感としては、バランスの取れた働き方をしていることを考慮すると、まずは創造的なエネルギーでこのプロジェクトをスタートさせ、その後すぐにチームを招集して洗練させ、拡張していくのが良いだろう。最終的な提出物は私に責任があるが、最高のアイデアは通常、グループから生まれる。

さて、「誰に最初に相談すべきか？」。まずは自分のチームだ。彼らはすぐそばにいて、様々な経験と働き方を持っている。64歳の佐藤は直感的で、誰も予想もしない、全く斬新で素晴らしいアイデアを思いつくかもしれない。23歳の伊藤も直感的で、本当に現代的で新鮮な視点をもたらしてくれるだろう。そして、鈴木、田中、高橋、渡辺のような思索的なタイプは、これらの最初の火花を取り上げて、どのように機能するか、どのようなメカニクスを構築するか、どのような深みを加えることができるかを考えることができる。良い組み合わせだ。

「このタスクにどのように取り組むか？」。おそらくチームミーティング、ブレインストーミングセッションから始めるだろう。自分のアイデアをあまり多く持った状態で臨むのは避けるつもりだ。なぜなら、まず他の人の考えを聞きたいからだ。私の役割は指示することではなく、促進することだ。タスクの広範で自由な性質を説明し、この段階では「悪いアイデアはない」ということを強調する。スポーツ要素、ファンタジー要素、RPG、あるいは全く新しいものを議論することができる。ゲームのコアとなる「もし〜だったら」という問いを定義する必要がある。

いくつかの緩いプロンプトを投げかけてみるかもしれない。
*   もしスポーツが勝つことではなく、生き残ることだったとしたら？
*   もしスポーツが完全に異質な世界で行われたとしたら？
*   もしアスリートがパフォーマンスに基づいて特別な力を得たとしたら？
*   あるいは、プレイヤーではなく、*究極*のスポーツアリーナを建設したり、ファンタジーリーグでチームを管理したりすることだったとしたら？

スポーツの競争性やスキルベースの性質と、強力な物語や世界構築要素を組み合わせたものを考えている。例えば、「未来的なグラディエーターのスポーツ」や「魔法を使って行うスポーツ」のようなものだ。

そして、軍事/ウォーゲーム企画部署の山口がいる。彼も34歳で、直感的な働き方をするリーダーだ。連携は必須ではないが、検討している。彼の部署のジャンルを考えると、全く異なる視点をもたらしてくれるかもしれない。スポーツの文脈の中で、軍事的な要素を刺激することはできないだろうか？ 例えば、ユニットを展開する戦略的なスポーツゲームや、古代の戦闘儀式から進化したスポーツのようなものだ。主要なステップではないが、内部のアイデアが動き出してから、外部から完全に新しい視点を得るための貴重な選択肢だ。壁にぶつかったり、いつもの枠組みから抜け出したいと感じた場合に、彼と簡単に話すのも悪くない。

ステップ1は、自分のチームとのオープンなブレインストーミングだ。彼らの多様な思考を活用できるようにする。特に、静かで思索的なタイプが安心して意見を共有できるようにする必要がある。次に、たくさんの生のコンセプトを得た後、物語、コンセプト、内容、スローガン、そして主要なセールスポイントをさらに開発するための有望な方向性1つか2つに絞り込んでいくことができる。きっと面白いことになるだろう。
