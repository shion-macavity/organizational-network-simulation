まず最初に、この「型破りな発想」を真剣に考えないといけない。軍事シューターの続編では、たとえそれが部署の得意分野だとしても、通用しないだろう。私は熟考するタイプなので、誰かに話す前に、アイデアをじっくりと煮詰める時間が必要だ。何も考えずに、最初に思いついたことを口に出したくない。

最初のアイデアは、軍事的な要素を、伝統的な戦争ゲームにしないようにどう捻じ曲げるか、だ。敵と戦うのではなく、構造、戦略、あるいは何か大きな出来事への*準備*についてかもしれない。あるいは、人間性、規律、連帯感、獲得したスキルを、全く異なる文脈に応用するのはどうだろうか？

普段は紛争に関連付けられるものを、協力的なものや創造的なものに変えるアイデアが好きだ。例えば、ライフシミュレーションやパズルゲームに戦略的な要素を組み合わせる、といった具合だ。例えば、「軍事戦略」を、災害救援のようなものに適用するのはどうだろう？あるいは、新しい惑星を開拓する、あるいは、大規模な人道危機におけるグローバルサプライチェーンを構築・管理する、といった具合だ。これは、計画、ロジスティクス、資源管理、チームワークといった軍事作戦の中核となる要素はそのままに、戦闘という目的からシフトさせる。

プレイヤーが兵士ではなく、戦略家や、特殊部隊を率いるエンジニアのような存在になるコンセプトを考えてみよう。「戦争」は他の人間ではなく、環境の課題や、調整された努力を必要とする複雑な問題に対して戦うものだ。

戦略ゲームによく見られる戦略的な深さと、ライフシムの創発的なストーリーテリング、あるいはパズルゲームの問題解決能力を組み合わせたものが欲しいと思っている。単なるゲームループではなく、強力な物語フックが必要だ。意味のあるストーリーは不可欠だ。

ストーリーとしては、破滅的な出来事の後、人類が新しい、敵対的な環境を再建し、テラフォーミングしなければならない世界を舞台にできる。軍事的な要素は、さまざまな側面を扱う高度に組織化された特殊チームから生まれる。エンジニアリング、資源収集、科学研究、そして*自然*からの脅威に対する防衛（人間ではない）。重点は、生存、協力、そして長期的な計画に置かれる。「戦争」は自然との戦い、「キャンペーン」は人類の未来のためのものだ。

コンテンツとしては、さまざまな基地を管理し、特殊部隊を配置し、環境に適応するための新しい技術を研究し、プレッシャーの下で重要な決定を下すことになる。ゲームプレイには、展開と資源管理のためのリアルタイム戦略要素と、乗組員の士気と能力に影響を与える、より親密でキャラクター主導の物語上の選択肢を組み合わせることができる。複雑なエンジニアリングの課題のためのパズル要素もあるかもしれない。

スローガンとしては…挑戦と希望の両方を捉えるものだ。「再建。再建。記憶せよ。」あるいは、「昨日の戦いではなく、明日のための戦い。」壮大でありながら、人間の努力に基づいたものにな感じられる必要がある。

主なセールスポイントは次のとおり。「伝統的な戦闘のない、深い戦略的計画」、「人類の未来に影響を与える意味のある選択」、「戦略、シミュレーション、物語主導の進行のユニークなブレンド」、「協力が究極の武器となる世界でのリーダーシップを体験」。

よし、これは自分の内部処理としては良いスタートだ。これらの考えをもう少し洗練させ、プレイヤーが直面する課題の種類など、具体的な例を検討してみるかもしれない。これを頭の中で少し明確にしたら、これらのアイデアをいくつか試してみるつもりだ。まず吉田さんや加藤さんに話してみるのがいいだろう。彼らも熟考するタイプなので、より発展したアイデアを共有できるかもしれない。ヤマグチさんはリーダーなので、より洗練されたバージョンを彼に伝えることにする。そして、直感的な小林さんは、後で何かを刺激するようなワイルドなアイデアを持っているかもしれないが、今は自分の道を切り開く必要がある。まだ会話に飛び込む必要はない。
