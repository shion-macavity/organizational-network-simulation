## ■ ターン 0
* **【初期思考】**
> さて、ブレインストーミングを始めましょう。新しいゲームプロジェクト、ですか。しかも「ホラー」の範疇でも、予想外のものを考えてほしい、と。この手の挑戦は、今の私にはちょうどいいんです。これまでの経験から、真の革新は、往々にして期待を裏切ることから生まれると知っているからです。

まず最初に、どうアプローチするか、ですね。私の仕事のスタイルは直感的、というやつです。ですから、最初に厳格なフレームワークや市場データとにらめっこするのは得意じゃないんです。一番良いアイデアは、たいてい突然、ひらめくんです。まるで火花が散るようで、それをたどっていく。だから、少し静かな時間が必要ですね。散歩でも、天井をじっと見つめて、思考をさまよう時間でも。思いつく限りの単語やイメージ、感情を書き留めるくらいでしょう。「ホラー」が部署のテーマですが、完全にジャンル外でもいい、と言われた。それは解放感がありますね！もしかしたら、最初はホラーらしくないホラーゲームなのかもしれません。あるいは、古典的なホラーの定型をひっくり返すようなもの。期待を裏切るアイデアが好きです。

まずは、いくつかの初期コンセプトを頭の中でじっくりと煮詰めておこうと思います。他の人の意見に気を取られすぎたくないんです。それが私のやり方です。自分自身でコンセプトを確かめる必要があります。

さて、いくつかのアイデアができたら、誰に相談するのが最初でしょうか？

最終的には、山本さんにすべてが渡ることになるでしょう。彼女がリーダーで、全体をまとめ上げるからです。でも、未熟な考えをそのまま持ち込むのは避けたいんです。彼女も私と同じように直感的ですから、即興的なアイデアを評価してくれるかもしれませんが、少しだけ発展させた方が良いと思います。

田中さんと渡辺さんがいます。彼らも直感的です。きっと相性が良いでしょう。互いのアイデアを自由にぶつけ合いながら、発展させていけるはずです。田中さんは47歳ですから、経験と柔軟な発想のバランスが取れているでしょう。あるいは、渡辺さんは少し若いが、それでも直感的で、私の最初の火花に新鮮な視点をもたらしてくれるかもしれません。そのような自発的で非構造的なブレインストーミングは、私にとって驚くべきブレークスルーにつながることがよくあります。判断せずに、アイデアを壁にぶつけて、何がくっつくか見てみましょう。それが、最初の「ひらめき」の後、アイデアを洗練させるための良い第一歩になると思います。

高橋さんは、思慮深い心の持ち主ですから、アイデアがもう少し固まったら、弱点を見つけ出して、実際に意味があり、実現可能かどうかを確認してくれるでしょう。磨き上げには貴重な存在ですが、最初の創造性の爆発には向いていません。

ですから、私の戦略はこうです。
1.  **一人時間:** 直感的な脳を自由にさまよう時間を取り、物語、コンセプト、潜在的なゲームプレイ要素に関する、大胆で制約のないアイデアを生成します。部署の「ホラー」要素に寄りかかりつつ、どのようにねじ曲げたり、他のジャンルと混ぜたり、ひっくり返したりするかを考えます。
2.  **非公式ブレインストーミング:** いくつかの初期の粗いコンセプトができたら、田中さんか渡辺さんを誘います。カジュアルな会話で、「こんなクレイジーなアイデアがあるんだけど、どう思う？」と聞いてみましょう。お互いの直感を活かし、より強力なコアコンセプトを一緒に構築できるかもしれません。そうすれば、自分の孤立した考えを提示するのではなく、すでに少し直感的な押し合いへし合いを経験したコンセプトを提示することになります。
3.  **後日:** ある程度まとまりがあり、ワクワクするようなものになったら、初めて山本さんに共有します。そして、さらに洗練が必要な場合は、高橋さんが、セールスポイントやキャッチフレーズを本当に鋭くするために、私を助けるのに最適な人でしょう。

他の部署は、今のところ気にしません。自分のチームには十分な頭脳がいるし、このプロジェクトは、まず自分の部署の創造的な限界を押し広げるものだと感じています。どんな奇妙なものを生み出せるか見てみましょう！たとえば…、本当に幽霊が出ているのではなく、そこに滞在した人々の記憶ゆえに幽霊が出ているかのように感じられる、古い日本の旅館。あるいは、植物自体が暗い秘密を抱えているガーデニングシミュレーター。ああ、もう頭の中が駆け巡っています！
