## ■ ターン 0
* **【初期思考】**
> 新しいゲームプロジェクトが始まり、ブリーフは「自由なアイデア」だ。普段はホラーを担当する私の部署にも。これは素晴らしい！ 型にはまらない考え方を奨励されるのは大好きだ。直感的な部分がすでに様々な可能性で活気づいている。

まず最初にすべきことは、本当にただ自分の心を自由にさまようことだ。まだ具体的なものを無理に作り出そうとはしない。ただ、制約がないときに「ホラー」とは何を意味するのかを広く考えてみる。もしかしたら、伝統的なホラーではなく、サスペンスフルで、不安を煽る、あるいは謎めいた核があり、それを発展させていけるようなものがいいかもしれない。まずは、何もフィルターをかけずに、本当に奇抜なコンセプトをいくつかブレインストーミングしてみるべきだ。たとえば、ゲームの奥深くまで進んで初めてホラーゲームだと感じさせるようなゲームはどうだろう？ あるいは、ゆっくりと忍び寄る恐怖感を持つライフシムはどうだろう？ 可能性は無限で、それがわくわくする。

さて、誰に最初に相談するかだが… 自由で制約のないアイデアを流そうとしているので、どんなに奇妙に思えても、コンセプトを本当に受け入れて、発展させてくれるような直感的な人に頼ると思う。そうすれば、お互いのエネルギーを活かし合い、本当にユニークなものにたどり着けるかもしれない。ワタナベかイトー、どちらも私と同じように直感的だから、最初の、遠慮のないブレインストーミングセッションには向いているだろう。実現可能性や構造を気にせずに、アイデアをぶつけ合える。サトウのようなバランスの取れたアプローチを持つ人は、もう少し後になって、いくつかの生のアイデアが出てきて、それを地に足の着いたものにするか、別の角度から見る必要があるときに相談するかもしれない。そしてもちろん、山本は私たちのリーダーだから、最終的にはより発展したコンセプトが必要になるだろうが、最初の火花を散らすためには、同じように直感的な頭脳の方が最適だと思う。

本当に楽しみだ。このような形でジャンルの制約を完全に無視できるのは、そう日常的なことではない。真に革新的なものが生まれるのはここだ。少し時間をかけて、さまざまな種類の恐怖、さまざまな種類の「不安」について考え、それをまったく予想外のゲームループとどのように組み合わせるかを考えてみるつもりだ。別の部署のものがアイデアのきっかけになるかもしれないが、まずは自分たちの部署内で始めるつもりだ。私のアプローチは、最初は非常に自由奔放になり、強力な核となるコンセプトや感情的な体験に焦点を当て、その周りにストーリーやメカニクスを構築していくことになる。
