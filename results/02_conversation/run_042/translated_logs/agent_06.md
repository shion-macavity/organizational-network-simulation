## ■ ターン 0
* **【初期思考】**
> さて、新しいプロジェクトだね。ゲーム全体をゼロからデザインするんだ – ストーリー、コンセプト、コンテンツ、スローガン、セールスポイント。そして完全に自由なアイデア出しで、それは素晴らしいけど、ちょっと…気が遠くなるね。私の部署は「スポーツゲーム企画」だけど、基本的には大まかなガイドラインで、アクションからライフシムまで、何でもありだって言われている。 「固定観念にとらわれず考えろ」って言われたけど、それは好きだ。頭の中がすでにアイデアでいっぱいだ。

まず最初に、リーダーとして、ボールを転がして、本当に斬新なアイデアが湧き出る最適な方法を見つけ出す必要がある。私の仕事のスタイルは直感的だから、硬直化したプロセスに陥りたくない。みんなが思いつく限りのあらゆることを気軽に思いつくように促したいんだ。悪いアイデアはない、ってやつだよ。後で洗練すればいい。

さて、誰に最初に話しかける？自分のチームは確実だ。それは当然だ。私たちの部署には色々な人がいる。佐藤、高橋、田中 – 彼らは私と同じように直感的だから、最初からたくさんの奇抜なコンセプトを生み出すのに適任だろう。鈴木はバランスが取れているのが良い。創造性を阻害することなく、ほどほどに私たちを地に足をつけていさせてくれる。それから、内省的な Watanabe と Ito がいる。彼らは後で、穴を見つけたり、詳細を考えたり、私たちの突飛なアイデアが実際に意味をなし、具体的なものに発展できるかどうかを確認するために重要になるだろう。しかし、最初のステップとしては、初期のコンセプトの爆発を得るために、直感的な側面に重点を置きたい。

この課題に最適なアプローチは、チームで大規模なブレインストーミングセッションから始めることだ。先入観も制限もない。みんなを会議室に集めて、お菓子を注文して、自由に話させればいい。彼らに「スポーツ」というラベルを無視して、どんなゲームなら自分が本当にプレイしたいか、どんな体験なら本当にユニークで楽しいか、自由に考えさせるように促したい。もし感じたら、後で「スポーツ」の要素を織り込むこともできるし、完全に捨てることもできる。会社は、それにこだわる必要はないと明言してくれているので、解放されている気分だ。

「もし世界が全く異なり、スポーツが原型をとどめないものに進化したら？」「もし平凡な活動が究極の競技スポーツになったら？」「あるいは、全く意味のないけど、信じられないほど楽しいゲームを作ったら？」というようなプロンプトから始めることを考えている。いつものサッカーやバスケットボールのコンセプトを超えて、彼らを突き動かしたいんだ。

たとえば、スポーツをプレイするだけでなく、ユニークなスポーツコミュニティの*生活*を生きるゲームを想像してみよう。もしかしたら、中心的な活動が、たとえば、競技ガーデニングと高速ホバークラフトレースの要素を組み合わせた奇妙で幻想的なスポーツである、協力型の「ライフシミュレーション」ゲームなのかもしれない。あるいは、スポーツのスキャンダルを舞台にした「ミステリー」ゲームだが、そのスポーツ自体は、リアルタイムの物理的要素と組み合わせた複雑な戦略的ボードゲームとしてプレイされる。それが、私が刺激したい「固定観念にとらわれず考えろ」という思考だ。

ヤマグチさん（ファンタジーゲーム企画部）については…彼もリーダーだし、彼の部署は私たちの部署とは全く違う。彼は内省的で、それは興味深いかもしれない。私は彼と直接つながっているのも良い。しかし、彼に*最初に*相談するつもりはない。私の優先順位は、自分のチームの創造性を刺激し、初期のアイデアを生み出すことだ。ある程度ブレインストーミングを行い、有望なコンセプトをいくつか絞り込んだ後で、ヤマグチさんと話すかもしれない。全く異なるジャンルの人からの外部の視点を得ることは良いかもしれない。彼は、私たち自身の考えに深入りしすぎて見落としていた何かを指摘したり、会社全体の戦略的な方向性に関する洞察を提供したりするかもしれない。しかし、それはステップ2か3であり、ステップ1ではない。

今のところ、チームを集めて、盛り上げて、アイデアを壁にぶつけることがすべてだ。本当にクールなものを作ろう。ブレインストーミングをスケジュールする必要がある。
