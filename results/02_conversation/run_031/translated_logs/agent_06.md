## ■ ターン 0
* **【初期思考】**
> さあ、山本、頭をフル回転させよう。新しいゲームのコンセプト、自由なアイデア、ストーリー、コンテンツ、キャッチコピー、セールスポイント…そして、クライム＆インベスティゲーションというジャンルはあくまで提案という考え方だ。気に入った。 「型破りな発想」は、もはや私のミドルネームだ。

さて、誰に話しかけて、どう取り組むべきか？

直感的なスタイルを考えると、まずアイデアを自由に流し出すのが良いと思う。超フォーマルな会議で、厳格な議題で突っ込むのは避けるべきだ。それは創造性を阻害するだけだ。まずは*最も*奇抜な可能性を探り、その後で絞り込んでいく必要がある。

チームを見回すと、深く掘り下げて洗練させるのが得意な思慮深いメンバーが多いのは素晴らしい。しかし、最初の「もし～だったら」というエネルギーを生み出すためには、私のブレインストーミングのスピードに追いつける人が必要だ。それが、私をまっすぐに向かわせるのは伊藤だ。彼も私と同じように直感的で、若いため、境界線を押し広げるような斬新なアイデアを持っているかもしれない。私たちがお互いに刺激し合い、たくさんの風変わりなコンセプトを素早く生み出せる気がする。1時間話しているだけで、ホワイトボードを完全に異なるゲームタイプで埋め尽くせるだろう。ジャンルに関係なく。この最初の、フィルターのかかっていないアウトプットが、私にとって非常に重要だ。

「誰に最初に相談するか」としては、間違いなく伊藤だ。コーヒーを飲みながら気軽に話したり、会議室を借りてアイデアをスケッチしたりするのも良いだろう。プレッシャーもなく、悪いアイデアもなく、ただ純粋な想像力だけだ。

そして、「このタスクにどのように取り組むか」という一般的な方法としては、伊藤との最初のブレインストーミングの後、チーム全員を集めたい。しかし、再び、講義や報告会のように感じさせるようなやり方ではない。佐藤や鈴木のような思索的なメンバーも含め、誰もが自分の最も奇抜な考えを共有することに抵抗を感じないような環境を醸成したい。

伊藤と私が考え出したクレイジーなアイデアをいくつか共有することから始めるだろう。それらは決定的な方向性としてではなく、どれだけ遠くまで踏み込めるかの例としてだ。そして、全員に投げかける。「一瞬、ジャンルを忘れて。プレイヤーにどのような*体験*を提供したいのか？ どんな物語を語りたいのか？」 みんなで付箋を使ってキーワードや一文のコンセプトを書き出すような、批判のないセッションを行うのも良いかもしれない。

この段階の目標は、質より量だ。幅広い可能性を必要としている。ある程度の生のアイデアのプールができたら、その時になって初めてパターンが見え、最も有望なコンセプトを特定し、思慮深いメンバーがストーリーを肉付けしたり、メカニズムを調整したり、一貫性のあるビジョンを構築したりするのを手伝ってもらえる。その後、自然に合う場合は「クライム＆インベスティゲーション」の要素を織り込み、真に画期的なものがその範囲外で現れた場合は、完全に方向転換することもできる。

彼らに、物語の構造を考えさせたり、特定のゲームメカニズムがどのように機能するかを考えさせたり、潜在的な課題や独自のセールスポイントを特定してもらいたい。そして、その洗練されたコンセプトから、魅力的なキャッチコピーを作成し、主要な機能をリストアップする。

リーダーとしての私の主な役割は、エネルギーを高く保ち、誰もが発言するように促し、良いアイデアを早すぎる段階で潰さないようにしながら、最終的に一貫性があり、エキサイティングなものに落ち着けるようにすることだ。この初期段階は、堰を切ったようにアイデアを放出すること全てもくろっている。

だから、そうだ。まずは伊藤だ。騒がせよう！
