## ■ ターン 0
* **【初期思考】**
> さて、思考プロセスを始めましょう。ええと、私は加藤、26歳、内省的で、現代設定ゲームプランニングで働いています。これは新しいゲームのアイデアを練る作業です——ストーリー、コンセプト、内容、スローガン、セールスポイント、全部ですね。そして、本当に何でも構いません。部門が「現代設定」とはいえ、かなりワクワクします。厳密なジャンルの制約もありません。

まず最初に思い浮かぶのは、内省的な性格なので、とにかく静かな時間が必要だということです。少なくとも頭の中にいくつかの初期の火花がないと、すぐさま話し出すことはあまりありません。同僚、特に経験豊富な人たちの前に、ただランダムな言葉を投げかけるのは、少し…準備不足に感じるかもしれません。他の人にアイデアをぶつける前に、頭の中だけでも少し構造を持っておきたいのです。

さて、このタスクにどう取り組むか？私の戦略は、まず個人的なブレインストーミングから始めることです。おそらくノートを取り出すか、空白のドキュメントを開いて、自由に連想を始めます。先入観をすべて払拭する必要があります。特に、型破りな考えをしろと言われているので。ゲームメカニズムでいつも試してみたいもの、心に響くストーリーテーマ、あるいは「話す動物」や「タイムトラベル探偵」や「音楽でできた街」のような、頭に浮かんだランダムなコンセプトを書き留めるかもしれません。この段階では、自己検閲はしないことが重要です。

ある程度の生のアイデアのプールができたら——たとえば、ストーリーのいくつかの非常に粗いコンセプト、潜在的なゲームプレイループがいくつか——それから誰に話すべきか考えます。まだ形のない考えをすぐに山口さんに持って行くのは避けたいです。彼はすべてをまとめる人なので、ある程度まとまったものを持って行きたいのです。

小林さんと佐々木さんは、どちらもかなり年上で経験も豊富です。私は彼らをとても尊敬しており、彼らの洞察は非常に貴重になるでしょうが、彼らに相談する前に、もう少し発展させておきたいと思っています。ゼロから助けを求めるのではなく、何かを考えていることを示したいのです。最初のブレインストーミングと中村さんとの会話の後、より確実なコンセプトがいくつかできたら、実現可能性や潜在的な落とし穴について彼らの意見を聞くかもしれません。

要するに：まず、自分自身に相談し、個人的にブレインストーミングします。次に、いくつかの初期のコンセプトを洗練するために、中村さんに気軽に、非公式なブレインストーミングセッションを依頼するでしょう。その後、1つか2つの強力なアイデアを具体化し、潜在的に小林さんや佐々木さんに、より経験豊富なフィードバックを求めることになるでしょう。
