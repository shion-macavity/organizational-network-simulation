## ■ ターン 0
* **【初期思考】**
> さて、どうしましょう… 全く新しいゲームのコンセプト、ゼロから作り上げる、ということで。「固定観念にとらわれない」発想で、私の番です。まさに、この手の制約のない課題は大好きです。ルールがないと、頭の中でアイデアがどんどん湧いてくるんです。

さて、誰に話しかければいいのか、そもそもどこから始めればいいのか。

私の仕事のスタイルからすると、いつものようにアイデア出しから突っ込み始めるのが普通です。この段階で構造を深く考え込む必要はありません。細部や実現可能性を気にする前に、まずは初期の、生っぽいコンセプトをいくつか出しておきたいんです。それが楽しいところでしょう？ ただ、想像力を自由に羽ばたかせるんです。

誰に相談すべきか… そうですね、私と同じように、自由な発想で直感的にブレインストーミングを楽しめる人を探したいです。突飛なアイデアを気軽にぶつけ合えるような、すぐに批判してこない人。どうするかは後で考えればいいんです。部署の中では、中村が思い浮かびます。彼も直感的な仕事のスタイルで、しかもリーダーではないので、より気軽で、まともなクリエイティブセッションの雰囲気が出ます。コーヒーでも飲みながら、壁にパスタをぶつけて何がくっつくか試してみるような感じで、気軽に話し合ってみるのがいいでしょう。時々、一番クレイジーなアイデアが何か素晴らしいものを生み出すことがあります。それに、彼は若いので、トレンドや若いプレイヤーがどんな体験を求めているかについて、新鮮な視点を持っているかもしれません。それは常に良いことだと思います。

アプローチとしては、間違いなく広範囲から始めるでしょう。とてつもなく広範囲に。具体的なメカニクスではなく、*感情*や*体験*について考えることから始めます。例えば、「プレイヤーにどんな感情を味わってほしいのか？」。驚き？興奮？穏やかな気持ち？スリリングな謎？ 核心となる感情的なフックが決まったら、その周りに世界や物語を構築していくことができます。

「現代設定」ですが、何でも取り入れられるということなので、馴染みのあるものと幻想的なものを混ぜたり、日常にちょっとしたひねりを加えたりすることを考えています。完全にありふれたものに、何か特別なものを注入するのはどうでしょうか？あるいは、現代的な設定の中に古代の秘密が隠されていたり、古い伝統が重要な未来が訪れたりするというのはどうでしょうか？そういう対比は、私にとって面白いアイデアの源泉となることが多いです。

とにかく、「もし〜だったら」という可能性のあるシナリオをたくさん生み出したいと思っています。「もしあなたのスマートホームが自我を持ち始めて、あなたの生活をさりげなく邪魔し始めたら？」とか、「もし都市のすべての街灯に、アンロックできる忘れられた記憶が隠されていたら？」とか、「もし料理番組が、実は地下の諜報組織のフロントだったとしたら？」みたいな。ほら、アイデアを自由に流しているだけです。

それから、これらの初期の突飛なコンセプトの中から、中村と私が最も魅力的だと感じるものをいくつか選び出し、大まかなストーリーの展開、主人公はどんな人物か、そしてコアとなるゲームループはどのようなものになるかをスケッチしてみます。もちろん、まだ非常に大まかなレベルの話です。この段階では、厳密なルールや技術的な制約は必要ありません。ただ、純粋な想像力です。

いくつかのしっかりとしたコンセプトが得られたら、*その時*に山田さんや加藤さんのような人に相談することを考えます。彼らは思慮深いので、これらの生っぽいアイデアを取り、論理的な構造を考えたり、潜在的な問題を特定したり、よりまとまりのあるものに洗練させる方法を提案したりするのに適しています。しかし、まずは点火、洗練ではありません。

だから、まず最初の一歩は、中村を見つけて、コーヒーを飲みながら自由にブレインストーミングすることです。アイデアを自由に流しましょう。セールスポイントやキャッチコピーは、コアコンセプトが強固になってから自然に生まれてくるものです。
