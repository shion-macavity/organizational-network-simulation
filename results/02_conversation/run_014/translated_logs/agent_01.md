## ■ ターン 0
* **【初期思考】**
> なんてこと、新しいゲームのコンセプトがあって、今回は本当に型破りな発想をしてほしいんだって！「自由なアイデアで、何でもあり」だって。それはわくわくするけど、ちょっと…気が重いかな。この年になると、色々なアイデアを見てきたからね。でも、良い挑戦はやっぱり好きだし、特に直感を自由に羽ばたかせるような挑戦は。

さて、誰に話しかけようか？まずはアイデアをいくつか打ち合わせてみたい。一人でじっと座っていると、まるで白いキャンバスを見つめているみたいだ。僕の最高のアイデアは、誰かと話しているとき、言葉を自由に流し、何が火花を散らすかを見ているときに生まれるんだ。

佐藤は、僕と同じように直感的で、それに若すぎない。良い基礎となるアイデアを持っているかもしれないし、少なくとも僕のより…型にはまらない考えを受け入れてくれるかもしれない。そして、渡辺、神様のご加護を。彼はとても若くて元気いっぱいで、きっと完全にぶっ飛んだ何かを思いつくだろう。それがまさに彼らが求めていることだ。もしかしたら、彼らの一人に手早くコーヒーでも飲んで、話のきっかけにしたい。特にフォーマルなことは何もいらない、気軽に話して、どんな最初の考えを持っているか見てみよう。高橋は思慮深く、後で洗練させるには素晴らしいけど、今は純粋でフィルターのかかっていないブレインストーミングが必要だ。そして、リーダーの山本には、少し形を整えた後で、最初のアイデアをいくつか持って行こう。ただの散らかった mess を持って行く意味はない。

僕のアプローチは、まず大枠から始めることだ。我々は「歴史ゲーム企画」だけど、厳密にそれにこだわる必要はないって言われた。つまり、歴史的な *テーマ* や *美学* をとって、完全にひっくり返してみるのはどうだろう？単に歴史を追体験するのではなく、歴史を改変したり、あるいは歴史的な *風味* を帯びた新しい歴史を作り出したりする。例えば、ファンタジーな歴史時代を舞台にしたライフシミュレーションゲームに、競争要素を加えたり、パズルを解くことで歴史そのものを変化させるパズルゲームとか。わからないけど、今はまだぼんやりしている。どんなにばかげたアイデアでも、まずは思いつくものを吐き出し、何が残るか見てみたい。ええ、渡辺や佐藤のような人と手早く話すのが完璧だ。僕たちの想像力がどこへ向かうか、見てみたいんだ。
